# 코드 구조

## Assets/Scripts 폴더 맵

```
Assets/Scripts/
├── Data/               ScriptableObject, CSV 데이터 모델
│   └── Record/         CharacterRecord, BettingData 등
├── Debug/              RaceDebugOverlay, 디버그 도구
├── Editor/             에디터 전용 (프리팹 생성기, StringTableValidator)
│   └── BettingUIPrefabCreator.cs
├── Manager/            씬 생명주기 관리
│   ├── GameManager.cs
│   ├── WalletManager.cs
│   ├── TitleSceneManager.cs
│   ├── BGMManager.cs
│   ├── SceneTransitionManager.cs
│   ├── UgsBootstrap.cs         (⏸ Phase 4)
│   └── UI/
│       ├── WaterDropDecor.cs
│       └── ...
├── Racer/              레이서 물리/로직
│   ├── RacerController.cs      ← 핵심
│   └── Collision/
├── UI/                 UI 컴포넌트
│   ├── BetAmountModal.cs
│   ├── CurrencyHeader.cs
│   ├── ExchangeModal.cs
│   ├── NicknameInputModal.cs   (⏸ Phase 4)
│   └── ...
├── Service/            외부 서비스 추상화
│   ├── LeaderboardService.cs   (⏸ Phase 4)
│   └── BadwordFilter.cs        (⏸ Phase 4)
└── Utility/            공통 유틸리티
    ├── OddsCalculator.cs       ← static, MonoBehaviour 아님
    ├── Loc.cs                  다국어 헬퍼
    └── ...
```

---

## SceneBootstrapper (partial 구조)

`SceneBootstrapper.cs`는 **partial class 6개**로 분할:

```
SceneBootstrapper.cs             기반
SceneBootstrapper.Betting.cs     배팅 패널 로직
SceneBootstrapper.Race.cs        레이스 시작/종료
SceneBootstrapper.Finish.cs      결과 화면
SceneBootstrapper.GameOver.cs    게임 오버 처리
SceneBootstrapper.GlobalRanking.cs  (⏸ Phase 5)
```

> ⚠️ 수정 시 **6개 partial 모두 함께** 검토 필수
