## OddsCalculator

**경로**: `Assets/Scripts/Utility/OddsCalculator.cs`

| 항목 | 내용 |
|------|------|
| 유형 | **static, MonoBehaviour 아님** |
| 역할 | 5가지 배팅 타입 배당 계산 |
| 클램프 | `Mathf.Max(1.1f, odds)` — 최소 배당 1.1 보장 |
| 정밀도 | `(int)Math.Ceiling(amount * (decimal)odds)` |

## CharacterVideoController

**경로**: `Assets/Scripts/Manager/UI/CharacterVideoController.cs` (SPEC-049)

| 항목 | 내용 |
|------|------|
| 역할 | 캐릭터 정보창 비디오 재생 래퍼 — `VideoPlayer → RenderTexture(640² Point) → RawImage` |
| 부착 | `IllustrationVideo` 노드 (RawImage 필수, `[RequireComponent]`·`[DisallowMultipleComponent]`) |
| 주요 API | `TryPlayForCharacter(CharacterData)`, 이벤트 `OnPrepared`/`OnFallback`, `StopAndRelease()` |
| 수정 규칙 | 무음(`audioOutputMode=None`)·루프·RectTransform 미변경 유지. **`Prepare()`는 활성 GameObject에서만** |

→ 상세 [[../시스템/라이브_포트레이트]]
