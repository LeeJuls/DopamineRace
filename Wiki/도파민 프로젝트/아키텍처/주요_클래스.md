# 주요 클래스 & 규칙

> ⚠️ 각 클래스에는 **수정 시 반드시 지켜야 할 규칙**이 있다.

---

## RacerController

**경로**: `Assets/Scripts/Racer/RacerController.cs`

| 항목 | 내용 |
|------|------|
| 역할 | 레이서 물리 이동, V4 스탯 계산, HP 드레인 |
| 수정 규칙 | **반드시 `RaceBacktestWindow`도 함께 수정** |
| V4 메서드 | `InitV4()`, `UpdateV4()`, `CalcSpeedV4()`, `ConsumeStaminaV4()` |

```csharp
// 구간 판정
bool IsInBurstZone()  // ⚠️ 버그 수정 후 이 메서드 기반 판정 (2026-03-20)
```

## OddsCalculator

**경로**: `Assets/Scripts/Utility/OddsCalculator.cs`

| 항목 | 내용 |
|------|------|
| 유형 | **static, MonoBehaviour 아님** |
| 역할 | 5가지 배팅 타입 배당 계산 |
| 클램프 | `Mathf.Max(1.1f, odds)` — 최소 배당 1.1 보장 |
| 정밀도 | `(int)Math.Ceiling(amount * (decimal)odds)` |

## CharacterRecord

**경로**: `Assets/Scripts/Data/Record/CharacterRecord.cs`

| 항목 | 내용 |
|------|------|

| charId 형식 | `char.type.name.NNN` |

## WaterDropDecor

**경로**: `Assets/Scripts/Manager/UI/WaterDropDecor.cs`

- UI 패널 물방울 장식 컴포넌트
- `vfx_circle` 재사용, 행별 알파 점감, `siblingIndex = 1`
- **Canvas overrideSorting**: 에디터 프리팹 생성 시 `SerializedObject` 필수 (직접 프로퍼티는 저장 안 됨)
