# 배팅 시스템

> **상태**: ✅ 완료  
> **핵심**: 5가지 배팅 타입 × 배당 계산 × BetAmountModal 입력 플로우

---

## 배팅 타입 (5종)

| 타입 | 설명 | 난이도 |
|------|------|--------|
| 단승 | 1위 맞추기 | ★ |
| 복승 | 1·2위 순서대로 | ★★★ |
| 연승 | 1·2위 (순서 무관) | ★★ |
| 삼복승 | 1·2·3위 순서대로 | ★★★★★ |
| 삼연승 | 1·2·3위 (순서 무관) | ★★★★ |

---

## 배당 계산 (`OddsCalculator`)

```csharp
// 배당 최소값 보장 (spec028 R6)
float finalOdds = Mathf.Max(1.1f, calculatedOdds);
```

- `OddsCalculator`: **static, MonoBehaviour 아님**
- 배당이 1.1 미만으로 계산되어도 UI에 1.1로 표시 (옵션 차단 X)
- 오버플로우 방지: `int reward = (int)Math.Ceiling(amount * (decimal)odds)`

---

## 베팅 플로우

```
BettingPanel
  └─ 캐릭터 선택 (배지 연동 — spec037)
        └─ 배팅 타입 선택
              └─ START 클릭
                    └─ BetAmountModal 팝업
                          ├─ 베팅액 입력 (슬라이더 또는 직접 입력)
                          └─ 확정 → 레이스 진행
```

---

## BetAmountModal (spec028 Phase 2)

- START 클릭 시 모달 팝업
- 보유 젤리 이내에서 베팅액 선택
- 배당 미리보기 표시 (예상 획득 젤리 + 스톤)
- Canvas sortingOrder 1000으로 다른 UI 위에 표시

---

## 보상 계산

```
적중 시:
  획득 젤리 = Mathf.CeilToInt(베팅액 × 배당)
  획득 스톤 = 베팅액 × 1  (고정 비율)

빗나감 시:
  젤리 전액 손실
  스톤 변화 없음
```

→ 통화 처리: [[통화_시스템]]

## 고양이 장식 (spec038)

배팅 화면에 고양이 장식 추가:
- `OnMouseDown` 가드: 레이스 중 클릭 차단
- Canvas sortingOrder 조정으로 배팅 UI 위 표시
