## StringTable.csv

**경로**: `Assets/Resources/Data/StringTable.csv`

- 열 구조: `UID, ko, en, jp, cn, de, es, br, tw` (9열)
- 신규 키 추가 시 **8개 언어 모두** 입력 (영어 fallback 허용)
- 현재 422개 데이터 행 (헤더 포함 424줄)

## 키 네이밍 규칙

```
str.{영역}.{기능}.{항목}

# 아이템(SPEC-050) — 향후 표준
str.dopamine.item.{name|desc}.{아이템id}
  예: str.dopamine.item.name.dopaminejelly / str.dopamine.item.desc.dopaminestone

예:
  str.bet.modal.title         배팅 모달 제목
  str.currency.jelly.name     젤리 통화 이름
  str.gameover.title          게임 오버 화면
  str.rank.global.title       글로벌 랭킹 제목
  str.char.001.name           001번 캐릭터 이름
  str.hud.goal                HUD 목표 텍스트 (⚠️ 삭제 금지)
  str.hud.start               HUD 시작 텍스트 (⚠️ 삭제 금지)
```

## 사용법

```csharp
// 기본 사용
string text = Loc.Get("str.bet.modal.title");

// 동적 키 조합 (간접 참조)
string charName = Loc.Get($"str.char.{charId}.name");
// → 해당 범위 키 일괄 발급 필수
```

## 검증 도구

**`DopamineRace > Validate StringTable Keys`**

- 컴파일 완료 시 + Play 직전 자동 실행
- 누락 키 (코드 O, CSV X) 검출
- 미사용 키 (CSV O, 코드 X) 검출
- 커밋 전 반드시 누락 키 0건 확인

## 폰트 시스템 (SPEC-048)

### 언어별 폰트 라우팅

| 언어 | 폰트 | 경로 |
|------|------|------|
| ko | PFstardust (koreanFont) | GameSettings.koreanFont |
| en·jp·de·es·br | PFstardust (mainFont) | GameSettings.mainFont |
| **cn** | **Fusion Pixel zh_hans** | GameSettings.chineseSimplifiedFont |
| **tw** | **Fusion Pixel zh_hant** | GameSettings.chineseTraditionalFont |

### Fusion Pixel Font (SPEC-048 도입)

- **출처**: github.com/TakWolf/fusion-pixel-font
- **라이선스**: SIL OFL 1.1 (상업적 무료) — `Assets/Resources/Fonts/OFL.txt` 동봉
- **파일**: `Assets/Resources/Fonts/fusion-pixel-12px-proportional-zh_hans.ttf` · `...zh_hant.ttf`
- **임포트 설정**: Dynamic + HintedRaster (픽셀 크리스프)
- **폴백**: GameSettings 미지정 시 Resources.Load로 직접 로드 (FontHelper 캐시)

## 서수 + 접미사 패턴

서수 레이블은 포맷 키 재활용:

```
str.bet.label.rank_bet = "{0} 배팅"
+ 기존 str.rank.first / str.rank.second 등과 조합
```

## 신규 키 추가 절차

1. `StringTable.csv`에 키 추가 (8개 언어 입력)
2. 코드에 `Loc.Get("str.xxx")` 사용
3. `Validate StringTable Keys` 실행 → 0건 확인
4. 커밋

> 위 순서 중 하나라도 빠지면 커밋 금지
