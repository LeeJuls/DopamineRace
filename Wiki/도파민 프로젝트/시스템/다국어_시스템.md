# 다국어 시스템

> **상태**: ✅ 완료 (7개 언어 지원)  
> **철칙**: 모든 문자열 하드코딩 절대 금지

---

## 지원 언어 (7개)

`ko` · `en` · `ja` · `zh_CN` · `de` · `es` · `br`

---

## StringTable.csv

**경로**: `Assets/Resources/Data/StringTable.csv`

- 열 구조: `key, ko, en, ja, zh_CN, de, es, br`
- 신규 키 추가 시 **7개 언어 모두** 입력 (영어 fallback 허용)

---

## 키 네이밍 규칙

```
str.{영역}.{기능}.{항목}

예:
  str.bet.modal.title         배팅 모달 제목
  str.currency.jelly.name     젤리 통화 이름
  str.gameover.title          게임 오버 화면
  str.rank.global.title       글로벌 랭킹 제목
  str.char.001.name           001번 캐릭터 이름
  str.hud.goal                HUD 목표 텍스트 (⚠️ 삭제 금지)
  str.hud.start               HUD 시작 텍스트 (⚠️ 삭제 금지)
```

---

## 사용법

```csharp
// 기본 사용
string text = Loc.Get("str.bet.modal.title");

// 동적 키 조합 (간접 참조)
string charName = Loc.Get($"str.char.{charId}.name");
// → 해당 범위 키 일괄 발급 필수
```

---

## 검증 도구

**`DopamineRace > Validate StringTable Keys`**

- 컴파일 완료 시 + Play 직전 자동 실행
- 누락 키 (코드 O, CSV X) 검출
- 미사용 키 (CSV O, 코드 X) 검출
- 커밋 전 반드시 누락 키 0건 확인

---

## 필수 보존 키 (삭제 금지)

| 키 | 이유 |
|----|------|
| `str.hud.goal` | `BuildTrackProgressBar`에서 참조 |
| `str.hud.start` | `BuildTrackProgressBar`에서 참조 |

---

## 서수 + 접미사 패턴

서수 레이블은 포맷 키 재활용:

```
str.bet.label.rank_bet = "{0} 배팅"
+ 기존 str.rank.first / str.rank.second 등과 조합
```

---

## 신규 키 추가 절차

1. `StringTable.csv`에 키 추가 (7개 언어 입력)
2. 코드에 `Loc.Get("str.xxx")` 사용
3. `Validate StringTable Keys` 실행 → 0건 확인
4. 커밋

> 위 순서 중 하나라도 빠지면 커밋 금지
