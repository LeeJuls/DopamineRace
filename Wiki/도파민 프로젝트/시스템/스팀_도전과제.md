# 스팀 도전과제

> AppID **4532310** · 기본 도전과제 **10종** · `DR_STEAM` 조건부 컴파일.
> SPEC-046(스캐폴드) → 2026-06-27 **연동 완성**(임포트 + 심볼 + 실증).

---

## 아키텍처

| 컴포넌트 | 역할 |
|----------|------|
| `Assets/Scripts/Steam/SteamManager.cs` | Steam API 수명주기. `RuntimeInitializeOnLoadMethod`로 자동 부트스트랩(씬 배치 불필요), `DontDestroyOnLoad`, `SteamAPI.RunCallbacks` |
| `Assets/Scripts/Steam/SteamAchievements.cs` | 해금 진입점 `Unlock(apiName)`. **멱등**(GetAchievement 가드 후 SetAchievement→StoreStats) |
| `com.rlabrecque.steamworks.net/` | Steamworks.NET 2025.163.0 코어 (asmdef `autoReferenced: true`) |

- **API Name은 내부 식별자(비노출)** → StringTable 키 발급 대상 아님. 사용자 노출 이름/설명은 파트너 사이트에서 언어별 입력(VDF 10개 언어 Publish).
- `DR_STEAM` 미정의 시 전체가 안전한 no-op 스텁 → 빌드 안 깨짐.

## 도전과제 10종 (훅 위치)

| API Name | 조건 | 훅 |
|----------|------|----|
| `ACH_FIRST_BET` | 첫 배팅으로 레이스 시작 | `GameManager.StartRace` |
| `ACH_FIRST_WIN` | 첫 적중 | `GameManager.CalcScore` |
| `ACH_EXACTA_HIT` | 쌍승(1·2등 순서) | `GameManager.CalcScore` |
| `ACH_TRIO_HIT` | 삼복승(1~3등) | `GameManager.CalcScore` |
| `ACH_BIG_WIN` | 한 라운드 스톤 **500+** | `GameManager.CalcScore` |
| `ACH_RICH` | 누적 스톤 **3000+** 도달 | `WalletManager.Reward` |
| `ACH_CAT_POWER` | 고양이의 힘 첫 사용 | `WalletManager.TryUseCatPower` |
| `ACH_RESCUE` | 구제(파산 직전 생존) | `WalletManager.TryRescue` |
| `ACH_FULL_CLEAR` | 전 라운드 완주 | `GameManager.ChangeState(Finish)` |
| `ACH_GAME_OVER` | 첫 파산 | `GameManager.ChangeState(GameOver)` |

> 임계값 500/3000은 [[시스템/통화_시스템]] SPEC-047(스톤 기준) 개정 반영. 모두 세션 내 즉시 판정 + 멱등.

## DR_STEAM 빌드 규칙 ⚠️

- **Standalone 타겟 전용**. Steamworks.NET asmdef `includePlatforms`에 WebGL 없음 → **WebGL에 `DR_STEAM` 켜면 컴파일 에러**.
- `PlayerSettings.SetScriptingDefineSymbols(NamedBuildTarget.Standalone, ...)`로 추가 (기존 `AMPLIFY_SHADER_EDITOR` 보존).
- `steam_api64.dll`/`steam_api.dll`은 `.gitignore` 미적용 → 커밋 포함(빌드 필수). `DopamineBuildVerify`가 DR_STEAM 빌드 시 dll 존재 검증.
- 프로젝트 루트 `steam_appid.txt`(`4532310`)는 에디터 Play 테스트용 — `.gitignore` 처리(로컬 전용).

## 테스트

- **자동(MCP)**: 훅 10/10, 상수 API Name 10/10, 임계값, DR_STEAM 컴파일 에러 0, Steam 미실행 시 Unlock 예외 0.
- **실증**: `[SteamManager] Steam 초기화 완료 (AppID 4532310)` + `ACH_FIRST_BET` 오버레이 팝업.
- **초기화**: 에디터 메뉴 `DopamineRace > Debug > Reset All Steam Achievements` (또는 `steam.exe -console` → `achievement_clear`).

## 알려진 사항

- **미출시 게임은 유저 프로필/커뮤니티 도전과제 페이지에 미표시**(Steam 정상 동작). 개발자 해금·팝업은 작동, 실제 카운트는 파트너 사이트 Stats & Achievements 확인. 출시 후 자동 노출.
- 누적 조건(N회 승리 등)은 Steam **Stat** 필요 — 현재 범위 밖.
