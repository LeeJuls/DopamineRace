# 타입별 전략

> 4가지 타입은 각각 **레이스의 어느 구간**에서 강한지가 다르다.  
> 서로 다른 타입이 각 구간에서 경쟁 → 순위 역전 발생 → 관람 재미 창출.

---

## Runner (도주) — 선두 질주형

| 항목 | 내용 |
|------|------|
| 부스트 구간 | **6 ~ 86%** (사실상 풀버스트) |
| 전략 | 선두를 잡고 HP를 태워 도주. 추격 차단. |
| 핵심 스탯 | Speed ↑↑, Accel ↑, Stamina 관리 |
| 약점 | 후반 HP 고갈 시 급격한 감속 |
| 특수 | `v4_runnerPersistentBurst`: 긴급부스트 목표 달성 전까지 지속 |

### 주목 캐릭터

| 이름 | 컨셉 | 총합 |
|------|------|------|
| 사일런스 | 완벽한 도주마 (에이스) | 114 |
| 적토 | 빠르면서 지치지 않는 이상적 도주 | 100 |
| 크리스탈 | 유리대포 (Sta 5, 종이 체력) | 90 |
| 씨비스킷 | 느린 도주 + Luck 20 = 로망 | 93 |
| 터프윈 | Spd 14의 느린 도주 + 체력 18 = 역설 | 97 |

## Leader (선행) — 균형 포지셔닝형

| 항목 | 내용 |
|------|------|
| 부스트 구간 | **26 ~ 45%** (초중반 포지셔닝) |
| 전략 | 도주마 뒤 슬립스트림 → 중반 추월 시도 |
| 핵심 스탯 | 균형, Intelligence ↑ (슬립스트림), Stamina ↑ |
| 약점 | 도주·추입에 비해 뚜렷한 극강 구간 없음 |

### 주목 캐릭터

| 이름 | 컨셉 | 총합 |
|------|------|------|
| 스페셜위크 | 만능 선행 엘리트 | 108 |
| 나리타 | 탱크 선행 (Pow 20) | 98 |
| 토카이 | Int 20 슬립스트림 마스터 | 93 |
| 파워블레이드 | 허약하지만 운빨 승부 | 90 |

---

## Chaser (선입) — 중반 폭발형

| 항목 | 내용 |
|------|------|
| 부스트 구간 | **46 ~ 65%** (중반) |
| 전략 | 중위권 대기 → 부스트 구간에서 선두권 진입 |
| 핵심 스탯 | Accel ↑↑, Intelligence ↑, 적절한 Speed |
| 약점 | 부스트 타이밍 미스 시 후순위 고착 |

### 주목 캐릭터

| 이름 | 컨셉 | 총합 |
|------|------|------|
| 딥임팩트 | 중반 폭발 최강 레전드 (Spd+Acc+Sta 모두 20) | 110 |
| 오구리캡 | 약점 없는 올라운더 | 98 |
| 라이스샤워 | Int 20 스포일러 (강자 저격) | 94 |
| 스칼렛 | Spd 20 유리 선입 (Sta 10) | 90 |

---

## Reckoner (추입) — 후반 역전형

| 항목 | 내용 |
|------|------|
| 부스트 구간 | **66 ~ 85%** (후반) |
| 전략 | 후미에서 체력 비축 → 후반 전력질주로 역전 |
| 핵심 스탯 | Stamina ↑↑, Power ↑, Accel ↑ |
| 약점 | 초반·중반 불리. 최종 스퍼트 전에 리드 좁혀야 함 |

---

## 밸런스 목표 (balance 에이전트 기준)

| 지표 | 목표 | 경고 기준 |
|------|------|----------|
| 타입별 1위율 | **10~13%** (4타입 균등) | 편차 >5% |
| 캐릭터별 승률 | 5~20% | >30% or <2% |
| 역전 발생 구간 | 70~100% | <5% or 초반 집중 |
| 트랙별 유리 타입 | 트랙마다 다른 타입 | 한 타입 50%+ |

→ 백테스팅 도구: `AutoRaceRunnerWindow`  
→ 수치 조정: `GameSettingsV4.asset`
