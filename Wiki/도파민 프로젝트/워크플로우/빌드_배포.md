# 빌드 & 배포

> 담당: [[에이전트_가이드|director(기획)·build(구현)·qa(검증)]]. Steam 출시 Windows 빌드.

---

## 빌드 타입 = IL2CPP (Mono 아님)
출시는 **IL2CPP**. Mono는 `Assembly-CSharp.dll` 즉시 디컴파일 → [[../시스템/보안_치팅방어|보안 Layer C]] + 성능 + Steam 표준 위해 IL2CPP. (현재 전환 완료)

| 설정 | 값 |
|------|-----|
| Standalone scriptingBackend | IL2CPP |
| managedStrippingLevel(Standalone) | **Disabled** (기록보관 직렬화 보호) |
| stripEngineCode | **0** (Cryptography/UnityWebRequest 보호) |
| Il2CppCompilerConfiguration | Release |
| bundleId / version | com.Dopamine.DopamineRace / 0.1.0 |

---

## 빌드 방법
**`Assets/Scripts/Editor/DopamineBuilder.cs`** (`[MenuItem]` + `BuildPipeline.BuildPlayer`).
- 메뉴 `DopamineRace/Build/`: Dev / Release / Release+SteamCopy / Verify.
- **batchmode**: `Unity.exe -batchmode -quit -projectPath <path> -executeMethod DopamineBuilder.CIRelease`
- `EnsureIL2CPPSettings`로 빌드 직전 백엔드/stripping/engineCode 강제 → 인스펙터 변경돼도 안전.

> ⚠️ **첫 IL2CPP 빌드는 MCP `unity_build` 금지** — 첫빌드 시간>MCP timeout → 좀비+산출물락([[MCP_사용법]]). 에디터 GUI 또는 batchmode로 1회 워밍 후 2회차+ incremental.

## 빌드 전 선결 (오너/수동)
- **AhnLab V3 예외** 등록: `Builds/`·`Library/Bee/`·`Library/il2cpp_cache/`·`%TEMP%` (빌드락 + HMAC/XOR 산출 exe 오탐 방지).
- IL2CPP 모듈 + VS C++ 빌드툴(빌드 PC마다). HOME 환경변수.

---

## 산출물 검증 (`DopamineBuildVerify`)
- 화이트(존재): `GameAssembly.dll`·`il2cpp_data`·exe·`UnityPlayer.dll`.
- 블랙(부재): `Assembly-CSharp.dll`·`MonoBleedingEdge/` (있으면 Mono = 전환 실패).
- **기록보관 회귀(최우선)**: IL2CPP 실행본에서 `leaderboard.json`+`.mac` 저장/로드(FP0)·PlayerPrefs StatsMAC·`Stone==제출점수`. **AOT 무음실패는 예외 안 던짐 → 데이터 직접 대조.** 빌드 실행본 검증은 전부 수동(MCP는 에디터 내부만).

## Steam 배포 (오너)
- App `4532310` / Depot `4532311` / 브랜치 `beta` / 계정 `clauzbt`.
- 빌드 → `C:\steamcmd\content\DopamineRace\`(메뉴 SteamCopy 또는 수동, `*_BurstDebugInformation_DoNotShip` 제외) + `steam_appid.txt`.
- `steamcmd.exe +login clauzbt +run_app_build C:\steamcmd\scripts\app_build_4532310.vdf +quit`.

## 규약 / 미결
- 산출물·`steam_appid.txt`·`*.mac` → gitignore. ProjectSettings는 커밋(설정 유실 방지).
- 미결: 실제 빌드+실행본 RG 검증(오너), stripping 최적화(`link.xml`), CI/CD(`game-ci`, Windows IL2CPP 러너+토큰 시크릿+steam-deploy).
