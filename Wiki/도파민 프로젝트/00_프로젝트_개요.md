# 도파민 레이스 — 프로젝트 개요

## 한 줄 정의

> **2D 픽셀아트 경마 배팅 게임** — 한정 시드(젤리 100)로 N라운드를 운용하는 로그라이크형 의사결정 게임. 누적 스톤으로 글로벌 랭킹 도전.

---

## 핵심 게임 루프

```
게임 시작 (젤리 100, 스톤 0)
    ↓
[배팅] 캐릭터 선택 + 베팅 타입 + 베팅액 결정
    ↓
[레이스] V4 시스템 — 5대 스탯 + 타입 전략 + HP 드레인
    ↓
[결과] 젤리/스톤 변동
    ↓
젤리 == 0 → GAME OVER (스톤 미저장)
마지막 라운드 완주 → 스톤 → 글로벌 랭킹 제출
```

**도파민 루프**: 짜릿한 배팅 → 역전 레이스 관람 → 파산 리스크 → 랭킹 경쟁

---

## 씬 구조

| 씬 | 인덱스 | 역할 |
|----|--------|------|
| `TitleScene` | 0 | 타이틀, 언어 선택, 캐릭터 퍼레이드, 랭킹 진입 |
| `SampleScene` | 1 | 게임 본편 (배팅 → 레이스 → 결과) |

---

## 게임 흐름 (확장)

```
TitleScene
  └─ START → SampleScene
  └─ 글로벌 랭킹 버튼 → GlobalRankingView (⏸ Phase 5)

SampleScene
  └─ BettingPanel (캐릭터 선택)
        └─ START → BetAmountModal (베팅액 입력)
              └─ 확정 → 레이스 진행
                    └─ 결과 → CurrencyHeader 갱신
                          ├─ 젤리>0 + 라운드 남음 → 다음 라운드
                          ├─ 젤리==0 → GameOverPanel
                          └─ 마지막 라운드 → FinishPanel → 랭킹 제출
```
