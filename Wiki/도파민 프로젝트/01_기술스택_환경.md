## 개발 도구

| 도구 | 용도 |
|------|------|
| **AutoRaceRunnerWindow** | N배속 × M회 자동 백테스팅 → 로그 저장 |
| **StringTableValidator** | 누락/미사용 Loc 키 검출 (`DopamineRace > Validate StringTable Keys`) |
| **BettingUIPrefabCreator** | 프리팹 재생성 (`DopamineRace > Create Betting UI Prefabs`) |
| **RaceDebugOverlay** | F4 디버그 오버레이 (HP바, 스탯 테이블) |
| **Unity MCP** | Claude에서 Unity 스크립트 실행/로그 읽기 |
