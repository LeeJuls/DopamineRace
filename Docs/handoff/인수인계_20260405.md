## 이번 세션(2026-03-29 ~ 04-05) 변경 사항

### SPEC-014 기준 7개 변경

1. **OverallProgress 서브WP 보간** (`RacerController.cs`)
   - WP 사이 Dot 투영으로 연속 progress 계산 (틱틱 순간이동 → 부드러운 이동)
   - `_maxOverallProgress` 단조 증가로 골인 시 드롭 방지

2. **트랙바 데이터 소스 교체** (`SceneBootstrapper.Racing.cs`)
   - `SmoothProgress` → `OverallProgress` (실제 위치 동기화)

3. **trackBarLerpSpeed 설정 추가** (`GameSettings.cs`, `GameSettings.asset`)
   - 기본값 `12f`, Inspector Range 0~30

4. **시작 WP 변경** (`RacerController.cs`)
   - `currentWP = 0` → `currentWP = 1`
   - WP0 = 결승선(스폰 측면), WP1 = 앞쪽 체크포인트

5. **타입별 스폰 정렬** (`RaceManager.cs`)
   - Runner→앞줄(0-2), Leader→앞~중, Chaser→중~뒤, Reckoner→뒷줄(6-8)

6. **컨디션이 HP풀에 미영향** (`RacerController_V4.cs`)
   - `v4MaxStamina *= condMul` 제거 → HP풀은 순수 STA 기반

7. **STA 최소 11 밸런스** (`CharacterDB_V4.csv`)
   - Crystal(003): STA 5→11 (SPD -3, ACC -3)
   - Secret(006): STA 8→11 (INT -1, LCK -2)
   - Amazon(004): STA 10→11 (INT -1)
   - Juhong(022): STA 10→11 (SPD -1)
   - Powerblade(015): STA 10→11 (LCK -1)
