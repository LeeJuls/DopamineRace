## 2. ⚠️ 다른 PC에서 가장 먼저 검증할 것

이번 세션 마지막 코드 변경은 **Unity 컴파일 검증 미완료** 상태에서 커밋됨
(작업 중 Unity 종료 → MCP 검증 불가). 다른 PC에서 Unity 열어 확인 필수:

| 검증 항목 | 방법 | 기대 |
|-----------|------|------|
| 컴파일 에러 | Unity 콘솔 | 0 에러 |
| 결과 화면 표시 | 레이스 1회 플레이 → 결과창 | "도파민 스톤 +N개 획득! (총: M개)" 잘림 없음 |
| 스톤 산식 라인 | 결과창 상단 | "베팅젤리 × 배당 = 스톤" (예: 19 × 1.4 = 27) |
| 스톤 = 배율 보상 | 적중 시 스톤 | `ceil(베팅젤리 × 배당)` 만큼 (베팅액 1:1 아님) |
