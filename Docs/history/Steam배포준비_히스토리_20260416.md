## 1. 패시브 스킬 시스템 구현

### 개요
기존 충돌 승리(CollisionWin)만 존재하던 스킬 시스템을 확장하여 Lap/HP/Rank 트리거와 다양한 효과 타입 추가

### 변경 파일
- `Assets/Scripts/Data/SkillData.cs` — 트리거/효과 타입 확장
- `Assets/Scripts/Data/PassiveSkillData.cs` — 패시브 스킬 데이터 클래스 신규
- `Assets/Scripts/Racer/RacerController.cs` — 패시브 스킬 런타임 상태 추가
- `Assets/Scripts/Racer/RacerController_V4.cs` — V4 스킬 발동 로직
- `Assets/Scripts/Racer/Collision/CollisionSystem.cs` — `IsCollisionSkillArmed` 연동
- `Assets/Resources/Data/CharacterDB_V4.csv` — 스킬 데이터 업데이트
- `Assets/Scripts/Data/CharacterDataV4.cs` — skillData 파싱 로직
- `Assets/Resources/Data/StringTable.csv` — 스킬 관련 문자열 추가

### 스킬 트리거 타입
| 타입 | 설명 |
|------|------|
| `E_Skill_Collision` | 충돌 N회 시 발동 (기존) |
| `E_Skill_Lap` | N랩 완주 직후 발동 (신규) |
| `E_Skill_HP` | HP N% 이하 도달 시 1회 발동 (신규) |
| `E_Skill_Rank` | N등 이하가 되는 순간 1회 발동 (신규) |

### 스킬 효과 타입
| 타입 | 설명 |
|------|------|
| `CollisionWin` | skillActive 중 충돌 무조건 승리 (기존) |
| `SpeedBoost` | outputSpeed 배율 증가 (상한 1.15) |
| `HpHeal` | 즉발 HP 회복 (상한 0.15) |
| `DrainReduce` | HP 드레인 배율 감소 (하한 0.55) |

## 2. Steam Windows 빌드 배포 준비

### 배경
v0.1.0 첫 Steam beta 브랜치 배포를 위한 환경 구성

### 완료 작업

#### ProjectSettings 수정
- Company Name: `DefaultCompany` → `Dopamine`
- Bundle Identifier: `com.DefaultCompany.2D-URP` → `com.Dopamine.DopamineRace`
- Version: `1.0` → `0.1.0`
- Scripting Backend: Mono → **IL2CPP**

#### .mcp.json 로컬 분리
- 개발 PC마다 Unity 프로젝트 경로가 달라 .gitignore 처리
- filesystem MCP 추가 (`@modelcontextprotocol/server-filesystem`)

#### SteamCMD 스크립트 생성 (로컬, 커밋 대상 아님)
- `C:\steamcmd\scripts\app_build_4532310.vdf`
- `C:\steamcmd\scripts\depot_build_4532311.vdf`
- App ID: 4532310 / Depot ID: 4532311 / 브랜치: beta

#### IL2CPP 모듈 설치
- **문제**: Unity Hub Validation Failed (AhnLab 백신 timeout)
- **해결**: 다운로드된 설치 파일 직접 실행
  - 경로: `%APPDATA%\UnityHub\downloads\UnitySetup-Windows-IL2CPP-Support-for-Editor-6000.3.7f1.exe`
- **결과**: `C:\Program Files\Unity\Hub\Editor\6000.3.7f1\Editor\Data\il2cpp\` 정상 설치 확인

### 미해결 사항

#### Unity PackageManager 에러
- Unity 에디터 직접 실행 시 `HOME` 환경변수 미설정으로 UPM 크래시
- `HOME = C:\Users\USER` 등록 완료
- Unity Hub를 통해 프로젝트를 열면 정상 동작
- **다음 단계**: Unity Hub → Projects → DopamineRace → 열기 → 빌드 실행

### 남은 작업
1. Unity Hub로 프로젝트 열기
2. File > Build Settings → Windows x64 IL2CPP 빌드 실행
3. 빌드 결과물 → `C:\steamcmd\content\DopamineRace\` 복사
4. SteamCMD 업로드 → Steamworks 대시보드 확인
