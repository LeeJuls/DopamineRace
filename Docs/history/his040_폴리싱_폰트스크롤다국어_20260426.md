## 완료 작업

### 1. 폰트 언어 기반 전체 적용

**문제**: CharacterInfoPopup 및 전체 씬 Text에 픽셀폰트 미적용.  
언어에 따라 다른 폰트를 써야 함 (KR → PFstardust3_0_ExtraBold, 그 외 → NeoDunggeunmoPro-Regular).

**수정 파일**:
- `Assets/Scripts/Utility/FontHelper.cs`
  - `GetFontForCurrentLanguage()` 메서드 추가 — `Loc.CurrentLang == "ko"` 분기
- `Assets/Scripts/Manager/UI/SceneBootstrapper.cs`
  - `Awake()` font init — `GetFontForCurrentLanguage()` 우선 사용
- `Assets/Scripts/Manager/UI/CharacterInfoPopup.cs`
  - `ApplyFontAll()` — `GetFontForCurrentLanguage()` 기반으로 모든 Text에 일괄 적용
  - `InitRadarChart()` / `UpdateRadarChart()` — `theme.common.font` 미리 설정 (플리커 제거)
  - `ShowSequence` 코루틴 끝에 `ApplyFontAll()` 추가 호출 (RadarChart 동적 Text 포함)
- `Assets/Prefabs/UI/BettingPanel.prefab`
  - 29개 Text 컴포넌트 전부 픽셀폰트 적용

### 2. 경기기록 스크롤 동기화 (단거리 · 중거리 · 장거리)

**문제**: 3개 RanksScroll이 독립 스크롤 — 같이 움직여야 함.  
`onValueChanged` 리스너 방식이 Unity 6 환경에서 발동 안 됨 → LateUpdate 폴링으로 교체.

**수정**: `CharacterInfoPopup.cs`
- `_shortScroll`, `_midScroll`, `_longScroll` 필드 추가
- `_lastShortH`, `_lastMidH`, `_lastLongH` 이전 위치 저장
- `LateUpdate()` — 매 프레임 horizontalNormalizedPosition 비교 → 변경 감지 시 3개 동시 설정
- BettingPanel.prefab — RanksScroll 3개: viewport=self, RectMask2D, horizontal=true, ContentSizeFitter(horizontalFit=PreferredSize), Text.horizontalOverflow=Overflow
