- 선행 1위율 **15.2%** (목표 10~13% 초과)

## 스탯 가중치 분석

실효 가중치 (SPD=100 기준):

| 스탯 | 공식 | 실효 가중치 |
|------|------|------------|
| SPD | Vmax = 2.5 × (1 + SPD × 0.02) | 100 |
| ACC | accelRate = ACC × 0.05 | ~50 |
| STA | MaxHP = 80 + STA × 3 | ~45 |
| INT | 패닉 감소 + 슬립스트림 + 버스트 타이밍 | ~35 |
| PWR | 충돌 시만 유효 | ~25 |
| LCK | P_크리티컬 = LCK × 0.005 / 3초 | ~5 |

32캐릭터 가중 점수 (최대 5,200): 1등 고요 5,010 / 꼴찌 비스킷 3,730 (격차 25%)

단순 합산 (최대 120): 1등 고요 114 / 꼴찌 봄빛 90 (격차 20%)

## Unity MCP 연결 이슈

이번 세션에서 `unity-code-mcp-stdio` MCP 툴이 로드되지 않음.

- 프로세스는 실행 중이었으나 (PID 7308, 22900 — 좀비 2개)
- MCP 툴은 Claude Code 세션 시작 시에만 로드되어 기존 대화에 반영 불가
- 해결: `Get-Process unity-code-mcp-stdio | Stop-Process -Force` 후 **새 세션**에서 시작

다음 세션 시작 전 체크리스트:
```powershell
Get-Process unity-code-mcp-stdio | Stop-Process -Force
# Unity 열린 상태에서 Claude Code 새 세션 시작
```
