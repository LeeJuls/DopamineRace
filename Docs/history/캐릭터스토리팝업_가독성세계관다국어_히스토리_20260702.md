# 캐릭터 스토리 팝업 가독성 + 세계관 정합 + 다국어 히스토리 (2026-07-02)

## 개요
캐릭터 정보창 펜 아이콘 → `CharacterStoryPopup`의 가독성 개선에서 출발해, 세계관 정본 확정·시나리오 정합 보강·32명 시나리오 8개 언어 번역까지 확장. 부수적으로 중국어 잭팟 모달 버그를 수정.

---

## 1. 스토리 팝업 가독성 (컬러 태그 + 문단 + 스크롤바)

### 표시 파이프라인 (`Assets/Scripts/UI/CharacterStoryPopup.cs`)
- **`FormatStoryText`** 신규: `ApplySemanticTags`(태그→`<color>`) → `§TIP§` 분리 → `GroupSentences`(2문장 문단, minChars 가드) → 팁 문단(파랑 이탤릭).
- **시맨틱 컬러 태그** (StringTable 콘텐츠에 삽입):
  | 태그 | 색 | 용도 |
  |------|-----|------|
  | `[skill]...[/skill]` | 골드 `#A67C00` + bold | 스탯 수치+효과 (숫자·% 있는 것만) |
  | `[key]...[/key]` | 오렌지 `#D9622B` | 출신세계 고유명사 + 소원 핵심구 |
  | `§TIP§`(마커) | 블루 `#3B6EA5` 이탤릭 | 배경/배팅팁 자동 분리 |
- **titleText 배선**: 프리팹 TitleLabel을 `titleText` 필드에 연결(국소 패치, factory 재생성 금지).
- 콘텐츠 가이드라인: 태그 중첩 금지·한 문장 내·쌍 필수·캐릭터당 2~5개.

### 우측 세로 스크롤바 (`CharacterStoryPopup.prefab`)
- 2계층(ScrollRect==viewport)을 **표준 3계층으로 승격**(ScrollRect / Viewport+RectMask2D / Content) — 신규 `CharacterStoryScrollbarPatcher.cs`(his046 패턴).
- `verticalScrollbar` 연결 + `AutoHideAndExpandViewport`(내용 길 때만). 핸들 하늘색(0.45,0.82,1)·폭 16px.
- 다국어로 텍스트 길어질 때(영어 +30~50%) 대응.

## 4. 32명 시나리오 8개 언어 번역

- **32명 × 7외국어**(en·jp·cn·de·es·br·tw) = 224셀 애니체 직번역. 020 꽃비 파일럿 → 4파벌 병렬 → C# 배치 반영.
- 규칙: 태그(`[skill]`/`[key]`/§TIP§)·수치 보존, 스탯 용어=`str.ui.char.stat.*` 일치, 캐릭터명=`str.char.NNN.name` 일치, cn 간체/tw 번체 구분, CSV 안전(ASCII 큰따옴표 회피→전각「」).
- **슬립 용어 통일**: 파벌별 불일치(滑行/打滑/Ausrutschen…)를 공식 `str.skill...slip`(气流/Sog/Rebufo/Vácuo/氣流)로 표준화.
- 검증: §TIP§ 264, 태그쌍 균형, 전 행 9필드, **리터럴 누출 0·한글 잔존 0** (execute_csharp 전수).
- 잔여: 새벽(032)은 `name` 키 없는 미완성 캐릭터 → 참조 시 Dawn/黎明 임의역, 완성 시 통일. 출시 전 네이티브 검수.
