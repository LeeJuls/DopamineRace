## 구현 (3종)

### A. `.claude/` 위생
- `settings.local.json` allow에서 RCE 와일드카드 12종 제거(→프롬프트화): powershell/python/start_process/execute_csharp/git push/reg delete 등.
- 공유 `.claude/settings.json`(신규·커밋) `deny` 블록 — 인터프리터-via-Bash + 임의프로세스 MCP. deny>allow라 전 PC 상속·재누적 방어.
- `.gitignore`에 `!.claude/settings.json` 화이트리스트.
- ※ `unity_execute_code`·`git push`는 deny 제외(검증·요청push 보존, 프롬프트로 충분).

### B. `CryptoSign.cs` (신규)
- `HmacHex(byte[]/string, key)` HMAC-SHA256 소문자 hex + `ConstantTimeEquals` + `LocalKey`(기기파생, 서버 비공유).

### C. `SecureInt.cs` (신규) + WalletManager
- 이중 마스크 사본(`value^maskA`, `value^maskB`) struct + `int` 암묵변환 → 호출부 무수정. default(SecureInt)=0 오탐 0.
- `WalletManager._jelly/_stone` → SecureInt. `Jelly`/`Stone` 프로퍼티 `int` 시그니처 불변. `#if UNITY_EDITOR` 디스플레이 미러로 인스펙터 표시 보존. `OnChanged?.Invoke()` → `RaiseChanged()`(미러 동기화 래퍼).

### D. 저장 HMAC
- `LeaderboardData`: `leaderboard.json` + `.mac` 사이드카(스키마/서버계약 불변). Load 시 검증, 불일치→캐시 무시. 레거시(MAC 없음) grace.
- `ScoreManager`: PlayerPrefs 누적스탯(score|rounds|wins) `DopamineRace_StatsMAC`, Awake 검증, fresh/legacy grace.
- CharRecords/BetRecords는 제외(odds용·마이그레이션 관리·FP 위험 큼·리더보드 무관).

- 컴파일 0(Phase B/C/D/E 각각) · 콘솔 에러 0.
