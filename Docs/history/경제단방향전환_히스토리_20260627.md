# 경제 단방향 전환 히스토리 (SPEC-047)

> 2026-06-27 · 젤리 소멸·스톤만 획득 · SPEC-035 되돌림

## 배경

오너가 "적중 시 젤리가 복리로 늘어 긴장감이 없다"는 점을 지적.
조사 결과 현재 동작은 버그가 아니라 **SPEC-035(2026-05-17) 오너 결정**(`stone=jelly=ceil(betAmount×배당)`)임이 확인됨.
이를 의도적으로 되돌려 단방향 경제(젤리 소멸, 스톤만 획득)로 개정.

## 작업 절차

1. **Stage 0** — 롤백 baseline 커밋+푸시 (`0c5297e`)
2. **client·qa 에이전트 검토** — plan 2차 업그레이드 (BetReward 리팩터, 모달·Finish 누락분, 변수 교체 필수 확인)
3. **Stage 1~4** — 단계별 개발 → Unity MCP 기능 테스트 게이트

## 핵심 결정 (오너)

| 항목 | 결정 |
|------|------|
| 스톤 공식 | `ceil(betAmount × 배당)` 유지 |
| 젤리 | 배팅 시 차감, 적중해도 반환 없음 |
| 도전과제 | BigWin 한 라운드 스톤 500+, RICH 누적 스톤 3000+ |
| 밸런스 | **범위 외** — catPowerUsesPerGame 등 미변경, 백테스트 없음 |
