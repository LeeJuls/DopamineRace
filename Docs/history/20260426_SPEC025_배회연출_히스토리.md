## 주요 이슈 및 해결

| 이슈 | 원인 | 해결 |
|------|------|------|
| ScatterAndWander 미호출 | GameManager.Start()가 RaceManager.Start()보다 먼저 실행 | RaceManager.Start()에서 기존 Betting 상태 체크 추가 |
| Walk 트리거 없음 | Animator에 Walk 미구현 | Sprint 0에서 확인 → Run + 속도 조절로 대체 |
| TC-2.4 7/9 (테스트 아티팩트) | 이전 세션 TC에서 StopWandering 후 재시작 안됨 | ScatterAndWander 재호출로 9/9 복구 확인 |
