## 3. 충돌 회피 luck → int 변경

### 배경
- 충돌 회피가 luck 기반 → "운 좋게 피했다"는 서사가 레이싱에 부자연스러움
- "영리하게 피했다"가 더 어울린다는 설계 검토

### 역할 분리
- **luck** = 크리티컬 발생 (확률적 이벤트)
- **int(지능)** = 상황 대응 (회피, 슬립스트림 활용)

### 변경
- `RacerController.cs` `TryDodge()`: `charBaseLuck × luckDodgeChance` → `charV4.v4Intelligence × gs4.v4_intDodgeChance`
- `GameSettingsV4.cs`: `v4_luckDodgeChance` → `v4_intDodgeChance` (Range 0.005~0.05)
- `GameSettings.cs`: 기존 `luckDodgeChance` Tooltip에 `[V4에서는 미사용]` 표기
- `CollisionSystem.cs`: 회피 로그 `(luck:X)` → `(int:X)`

## 5. 도주(Runner) 지속 긴급부스트 특성 추가

### 배경
- 기존 긴급부스트: 목표 순위 달성 시 즉시 해제 → 이탈 → 재발동 반복 (yo-yo)
- 도주 타입 특성상 앞에 있어야 하므로 목표 달성 전까지 끊지 않는 것이 자연스러움

### 설계 결정
- **제안(연속 유지)** vs **절충(정규 구간에서 강제 전환)** 검토
- **제안 채택**: 목표 순위 달성 전까지 긴급부스트 지속 → HP를 태워서라도 앞으로
- "나쁜 출발 = 도주의 리스크"로 배팅 전략에 직결되는 설계

### 변경
- `GameSettingsV4.cs`: `v4_runnerPersistentBurst` bool 필드 추가 (기본값 true)
- `RacerController_V4.cs` `UpdateV4EmergencyBurst()`:
  ```csharp
  // Runner + 설정 ON이면 목표 달성해도 정규 부스트 구간까지 유지
  bool isPersistentRunner = gs.v4_runnerPersistentBurst &&
                            charDataV4.charType == CharacterType.Runner;
  else if (!shouldEmergency && v4EmergencyBurst && !isPersistentRunner)
      v4EmergencyBurst = false;
  ```
- `RaceDebugOverlay.cs`: 긴급부스트 로그에 `도주지속:ON/OFF` 표시

### 효과 확인 (로그)
- 이전: Runner HP 많이 남기고 5~7착 빈번
- 이후: Runner HP 거의 완전 소진, 1~2착 달성 → HP 활용 효율 개선

## 6. 완주 기록 로그 순서 버그 수정

### 원인
- `RaceManager.OnRacerFinished` 콜백에서 `raceActive = false` 설정
- 직후 `LateUpdate`에서 `raceActive` 체크로 skip → 일부 완주 이벤트 미기록
- `SaveRoundReport`에서 누락분만 뒤에 추가 → 순서 역전 발생

### 증상
- 게임 UI(Rankings)는 정확한데 로그의 `완주 기록` 섹션 순서가 틀림

### 수정 (`RaceDebugOverlay.cs` `SaveRoundReport()`)
```csharp
// 기존: 누락분만 뒤에 추가 (순서 역전 가능)
// 수정: finishEvents를 finalRankings 기준으로 전체 재구성
var originalFinishEvents = new List<RaceEvent>(log.finishEvents);
log.finishEvents.Clear();
foreach (var entry in rankings)
{
    bool found = false;
    foreach (var fe in originalFinishEvents)
    {
        if (fe.description.Contains(entry.racerName))
        { log.finishEvents.Add(fe); found = true; break; }
    }
    if (!found) { /* 미기록분 새로 생성 */ }
}
```
