# 20260608 배팅 화면 배경 고양이 장식 히스토리

> 명세: `Docs/specs/spec038_고양이장식_배팅화면_20260608.md`

## 요약
Pet Cats Pixel Art Pack 고양이를 **배팅 화면(Betting)에서만** 사용자 지정 폴리곤 영역에 배회시키고, 경기 시작 시 despawn·복귀 시 재등장. 고양이별 **등장확률 SO** + **에디터 폴리곤 편집기**. client·qa harness 검토 + Phase 게이팅 TC 적용.

## 완료 작업
| # | 작업 | 내용 | 결과 |
|---|------|------|------|
| 1 | **데이터·유틸** | `CatAreaData`(폴리곤 Contains/RandomPointInside/Bounds) + `WeightedRandomHelper`(룰렛휠·비복원) | ✅ |
| 2 | **등장확률 SO** | `CatSpawnConfig`(종별 weight·spawnCount·frameInterval) + `CatPrefabFactory`(프리팹6+SO 자동생성) | ✅ |
| 3 | **편집기·배회** | `CatAreaEditor`(C토글/K저장/+−정점) + `CatController`(프레임순환·폴리곤 배회·flipX) + Track 배선 | ✅ |
| 4 | **상태연동** | `CatDecorationManager`(Betting spawn/그외 despawn·풀링·초기보정) + 매니저 배선 | ✅ |
