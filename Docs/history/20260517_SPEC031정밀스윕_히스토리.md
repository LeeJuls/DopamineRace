**날짜**: 2026-05-17 | **관련 SPEC**: spec031(정밀 스윕 절) | **결과**: 동결 유지(무변경)

## 흐름
- SPEC-033 통제 단일스탯 스윕 하네스로 정밀 INT 스윕 (과거 불가 사유 해소됨)
- 백업: `Docs/logs/GameSettingsV4.spec031sweep.bak`
- 그리드 shift{0.25,0.5,0.65}×floor{0.45,0.7,0.85}+이웃 → INT β 4타입 측정
  - 최적 일관 셀 (0.25,0.80): max/min 1.3×(현 ~2.6×), avg β 1.9, 전타입 β>0
- 회귀 가드(실스탯 전트랙 N=300 타입 σ):
  - (0.5,0.55 현동결)=σ2.52 ✅ 유일 안전
  - (0.25,0.80)=6.79 / (0.5,0.70)=6.01 / (0.5,0.85)=5.89 — 전부 FAIL
- 결론: INT 파라미터가 SPEC-032 타입경제와 강결합, σ-안전 개선점 부재
  → 백업 롤백(=현 동결 0.5/0.55 복원), asset 무변경. 정밀 재튜닝은 SPEC-032
  동시 재밸런스 필요(대규모) → 오너 결정 대기

## 교훈
