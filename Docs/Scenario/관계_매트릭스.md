## 비호감 관계 전체 목록

| 방향 | 이유 |
|------|------|
| 침묵 → 정의 | 이념을 레이스에 끌어들이는 것이 불필요해 보인다 |
| 정의 → 새벽 | 달리기 위해 달린다 — 이념 없음, 경박해 보임 |
| 정의 → 아마존 | 외계 규칙을 이념이라 부를 수 없다 |
| 골든데이 → 서풍 | 동경받는 것은 알지만 거만함이 불쾌하다 |
| 황금호 → 서풍 | 귀족적 드레인 전략이 비겁해 보인다 |
| 비스킷 → 광풍 | 분위기를 파괴하는 혼돈 에너지가 싫다 |
| 동풍 → 전마 | 음침하고 무거운 분위기가 불편하다 |
| 꽃비 ↔ 서풍 | 상호: 꽃비는 귀족 드레인을, 서풍은 마법사 전략을 경멸 |
| 봄빛 → 독수리 | 해적 기질 — 폭력적이고 무질서하다 |
| 홍염 → 세크레트 | 정보 장사로 레이스를 더럽힌다 |
| 백로 → 터프윈 | 무질서한 해적은 귀족 레이스의 격을 낮춘다 |
| 나리 → 젠 | 관찰만 하고 직접 뛰지 않는 것이 게으름으로 보임 |
| 인디밴드 → 레이디 | 관중 여론 게임이 음악의 순수성을 파괴한다 |
| 젠 → 홍염 | 감정으로 달리는 것은 데이터를 무시하는 행위 |
| 세크레트 → 홍염 | 홍염의 감정 노출이 정보 보안의 반대 |
| 터프윈 → 백로 | 규칙과 예의를 강요하는 귀족 태도 |
| 아마존 → 정의 | 이념 과잉, 매 레이스마다 설교한다 |
| 독수리 → 골든데이 | 선두 권위를 당연하게 여기는 태도 |
| 광풍 → 레이디 | 여론 계산 때문에 레이스 흐름이 꼬인다 |
| 레이디 → 광풍 | 예측 불가가 관중 여론 분석을 망친다 |
| 오캡 → 젠 | 너무 계산적이어서 함께 있으면 즐거움이 사라진다 |
| 천리풍 → 임팩트 | 차갑고 기계적인 태도, 강자의 오만 |
| 칵테일 → 아마존 | 외계 경찰 방식은 시장 논리를 무시한다 |
| 초원 → 독수리 | 해적 두목의 폭력적 압박이 싫다 |
| 크리스탈 → 골드런 | 슬립 전략은 계산 오류의 산물 (파벌 내 1쌍) |
| 골드런 → 파워블레이드 | 허무주의가 레이스 전략을 오염시킨다 |
