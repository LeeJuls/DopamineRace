# 세계관 B안 — 〈도파미나 제국〉 단일 국가

> 작성일: 2026-06-28 | 상태: 기획 초안 (A안 SPEC-040과 독립 분기)

---

## 1. 세계 개요

**캐치프레이즈: "달리기는 신성하다. 황제가 친히 보증한다."**

광활한 단일 대륙을 지배하는 〈도파미나 제국〉. 이 나라에는 전쟁도, 기근도, 외교 위기도 있었다. 그러나 황제가 즉위 첫날 선포한 것은 군사 계획도 경제 개혁도 아니었다. "달리기 대회를 국가 최고의 오락으로 선포한다." 제국 헌법 1조에 이 문장이 박혔고, 그날 이후 제국의 예산 우선순위는 완전히 바뀌었다.

수십 년간의 연구 끝에 제국 황립 마법과학원은 다중 세계 포털 기술을 완성했다. 처음에는 무역로 개척이 목적이었으나, 시범 탐사에서 황제의 눈을 사로잡은 것은 자원도 기술도 아니었다. "저 세계의 소녀들이 더 빠르다." 그날 이후 포털의 공식 용도는 바뀌었다. 제국 외무부는 다중 세계에서 달리기 소녀를 "외교적으로 확보"하는 부서가 됐다.

소환된 32명의 소녀는 각자의 세계에서 원하는 것을 지닌 채 제국에 도착한다. 소원, 귀환, 자유, 혹은 아무것도 원하지 않는 자. 어느 쪽이든 달리지 않는 선택지는 사실상 존재하지 않는다. 탈출 포털을 독점한 것은 제국이고, 탈출에 성공해도 돌아갈 세계는 포털 없이는 닿지 않는다. 다만 형무소가 생각보다 편안해서 일부러 탈출하지 않는 소녀도 있다는 것이 제국 당국이 공식 인정하지 않는 공공연한 비밀이다.

## 2. 도파민 젤리·스톤 시스템

### 2-1. 도파민 젤리 — 배팅의 물질화

관중이 레이스를 관람하며 느끼는 흥분, 기대, 좌절, 환호는 물리적 에너지로 전환된다. 이것이 도파민 젤리다. 제국은 이 원리를 발견한 이후로 "감정에 과세"하는 유일한 국가가 됐다.

| 항목 | 내용 |
|------|------|
| 생성 조건 | 관중의 배팅 행위 + 강렬한 감정 반응 |
| 주요 용도 | 포털 에너지원·소녀 소환 유지비·경제 유통 통화 |
| 발행 주체 | 황립 도파민 관리청 (황제 직할) |
| 희소성 | 레이스가 없으면 젤리가 생성되지 않음 → 제국 경제 = 레이스 경제 |
| 특이사항 | 젤리는 먹을 수 있음. 과하게 먹으면 과호흡 유발 (단맛이 매우 강함) |

### 2-2. 도파민 스톤 — 적중의 결정체

배팅이 정확히 적중할 때 도파민 젤리는 결정화 반응을 일으켜 고밀도 에너지 덩어리인 스톤이 된다. 젤리 대비 에너지 효율이 수십 배이며, 포털 개방에 필수적인 고급 재화다.

| 항목 | 내용 |
|------|------|
| 생성 조건 | 배팅 적중 순간의 감정 폭발 |
| 가치 | 젤리 1,000개 상당의 에너지를 스톤 1개에 압축 |
| 국가 역할 | 포털 운용 핵심 연료. 제국이 스톤 시장을 전량 매입 |
| 개인 역할 | 관중이 보유할 경우 귀족 작위 청원 자격 부여 (50개 이상) |

### 2-3. 경제 순환 구조

```
관중 배팅
  ↓ 감정 물질화
도파민 젤리 생성
  ↓ 배팅 적중 시
도파민 스톤 결정화
  ↓ 국가 매입
포털 에너지 확보
  ↓
다중 세계 소환 지속
  ↓
레이스 개최 가능
  ↓ (순환)
관중 배팅
```

> 핵심 논리: 레이스가 없으면 소환이 없고, 소환이 없으면 레이스가 없다.
> 제국의 국가 존립이 달리기 대회에 달려 있는 구조다.

## 4. 소환 계약 시스템

### 4-1. 계약 조건

소환된 소녀는 제국 도착 직후 "완주 계약서"에 서명한다. 서명하지 않아도 소환은 완료된 상태이므로 귀환 수단이 없다.

| 조항 | 내용 |
|------|------|
| 제1조 | 소녀는 지정된 레이스 시즌에 참가한다 |
| 제2조 | 완주 포인트 누적 시 소원 1건을 이행한다 (황제 직인 보증) |
| 제3조 | 계약 거부 시 제국 형무소 (이름: 도파민 가든) 입소 |
| 제4조 | 형무소 내 생활은 황립 기준 귀족 3등급 수준으로 보장 |
| 제5조 | 형무소 내 자발적 레이스 참가는 언제든 가능하며 포인트 정상 지급 |

### 4-2. 소원 이행 방식

| 소원 유형 | 이행 방법 | 이행 성공률 |
|----------|----------|------------|
| 귀환 (고향 세계 복귀) | 포털로 원래 세계 좌표 산출 후 개방 | 98% (좌표 오차 2% 존재) |
| 소원 성취 (출신 세계 내 사건 개입) | 포털로 귀환 + 황립 지원 물자 제공 | 75% (결과는 소녀 본인 역량에 달림) |
| 자유 (제국 내 거주) | 명예 시민권 + 귀족 3등급 저택 제공 | 100% |
| 기타 | 황제 직접 면담 후 결정 | 황제 기분에 따라 다름 |

### 4-3. 형무소 (도파민 가든) 실태

- 정식 명칭: 제국 도파민 인재 보호 시설
- 실내 온도 연중 22도 유지
- 3식 황실 조리사 담당
- 도서관·스파·개인 정원 구비
- 자유 시간 레이스 연습장 개방
- 유일한 제약: 문이 잠겨 있음

> 시즌 2 기준으로 계약 거부 후 형무소를 선택한 소녀 중 자발적으로 복귀 신청한 비율: 60%.
> 이유: "밖이 더 재미있어 보여서." 나머지 40%는 "여기도 나쁘지 않아서" 미복귀 상태.

### 3-4. 왜 소녀인가 — 황립 공식 입장

**도파민 파장 공명 이론** (황립 마법과학원 제47차 보고서)

도파민 젤리의 고품질 결정화에는 달리는 자의 "감정 의지 파장"이 관중의 "기대 흥분 파장"과 완전히 공명해야 한다.

| 달리는 대상 | 파장 공명 지수 | 결정화 결과 | 비고 |
|------------|-------------|-----------|------|
| 소녀 | 100% (기준값) | 순수 도파민 젤리 | 제국 표준 |
| 성인 남성 | 12% | 도파민 슬래그 (산업 부산물) | 규격 미달. 현재 도로 포장재로 전용 중 |
| 동물 | 0% | 없음 | 목표 의식 부재 → 공명 불가 |
| 어린이 (성별 무관) | 43% | 저등급 젤리 (단맛만 강함) | 먹기엔 좋으나 에너지 효율 낮음 |

> 황제 注: *"과학이 나의 심미안을 지지했다. 연구비 잘 썼다."*

연구 비고: 이 연구는 황제 취임 다음 날 발주됐다. 결론은 발주 당일 이미 황제가 구두로 통보했으며, 연구진은 3,000페이지 분량의 실험 데이터로 결론을 지지하는 작업을 47년간 수행했다. 반론 논문 2편이 존재하나 두 저자 모두 도파민 가든 입소 후 반론을 철회했다.

| **동물 레이서 실험** (황립 마법과학원, 비공개 시험) | 말 12두가 포털 프레임 돌파. 다차원 경계 균열 발생. 긴급 봉합에 스톤 5만 개 소모 | 동물 레이스 헌법 금지 (제8조). 말은 현재 황실 의전용으로만 보유 |
| **혼합 레이스** (도파미나우스 2세) | 관중 전원이 소녀 선수만 응원. 남성 선수 자발적 기권. 1레이스도 완주 못 함 | "혼합 레이스는 소녀 레이스다"라는 결론으로 종료 |

## 5. 관중 포지셔닝

### 5-1. 유저 = 제국 명예 관람 시민권자

유저는 다중 세계 출신임에도 불구하고 제국 황실 포고령 제77호에 의해 "명예 관람 시민권"을 부여받는다. 이 시민권은 제국 내 최하위 귀족보다 서열은 낮지만, 배팅 권한만큼은 귀족과 동등하게 보장된다.

| 권한 | 내용 |
|------|------|
| 배팅 참여 | 전 트랙·전 레이스 참가 가능 |
| 젤리 보유 | 본인 생성 젤리 50% 보유 (나머지 50% 국가 세금) |
| 스톤 수령 | 적중 시 스톤 직접 수령 |
| 관람석 등급 | 배팅 실적에 따라 상승 (일반석 → 귀족석 → 황실 보조석) |
| 정치적 권리 | 없음. 단, 황제가 마음에 들면 즉흥 작위 수여 가능 |

> 황실 포고령 77호 제3항: "명예 관람 시민권자가 배팅을 통해 생성하는 도파민 젤리는 제국의 국력이다. 따라서 이들을 환대하는 것은 황제의 의무이자 국가 안보 사항이다."

### 5-2. 유저의 제국 도착 경위

황립 도파민 관리청은 젤리 생성량 증가를 위해 "다차원 관람단 유치 사업"을 비밀리에 추진했다. 전략은 단순했다. 다중 세계에 포털로 전송 가능한 소형 홍보 쪽지를 무작위로 살포한 것이다.

> **[홍보 쪽지 전문]**  
> 〈당신은 다중 세계 달리기 대회 **무료 관람권**에 당첨되셨습니다〉  
> 포털 번호: [여기를 밟으면 자동 이동]  
> 주의사항: 없음  
> 반환 방법: 현장 안내 참조

쪽지를 밟는 순간 포털이 활성화되어 제국 입장 로비에 도착한다. 안내 담당자가 즉시 "명예 관람 시민권 서류"를 건넨다. "반환 방법"은 서류 37페이지에 기재되어 있다.

**귀환 조건: 도파민 스톤 1,000개 기탁 시 귀환 포털 1회 개방.**

### 5-3. 유저의 배팅 동기

| 동기 | 내용 |
|------|------|
| **귀환 목적** | 스톤 1,000개를 모아 귀환 포털 활성화. 이론적으로 가능 |
| **현실** | 배팅에 이기면 스톤이 쌓이고, 지면 젤리를 더 써야 한다. 어느새 스톤을 쓰는 것도 아깝다 |
| **제국의 의도** | 어차피 안 돌아간다. 돌아가려다 더 재미있어서 눌러앉는다 |
| **황실 포고령 77호의 진짜 의미** | "당신이 흥분할수록 제국 경제가 돌아간다" |

> 명예 관람 시민권자 중 귀환을 실제로 선택한 비율: 시즌 1 기준 7%.  
> 이유: "레이스 한 번만 더 보고 가려다가"

## 6. B급 연출 포인트

### 6-1. 황제의 진지함이 웃음이 되는 순간

| 상황 | 황제의 반응 |
|------|-----------|
| 소녀 1위 결승선 통과 | 황제가 자체 제작 트로피를 직접 시상. 트로피에 황제 친필 해설 멘트 새겨짐 |
| 비 트랙 기상 악화 | 경기 중단 요청을 거부. "비야말로 레이스의 참맛이다" 직접 우의 쓰고 해설 지속 |
| 소녀 탈출 시도 | 경비대 출동 전 황제가 먼저 확성기 들고 "어디 가려고? 오늘 레이스 남았어" |
| 외국 외교사절 방문 | 회담 30분 후 레이스 현장으로 이동. "이거 안 보면 외교 의미 없다" |
| 소녀 부상 | 즉각 황립 의료진 파견. 그리고 쾌유 카드에 "빨리 나아서 달려줘"라고 적음 |

### 6-2. 귀족들의 후원 경쟁이 빚는 촌극

| 귀족 | 후원 소녀 | 저지른 일 |
|------|---------|----------|
| 아쿠아 코스트 후작 | 비 트랙 담당 소녀 전원 | 방수 코팅 제복 독자 개발. 특허 출원. "후원이 아니라 R&D"라고 주장 |
| 블리자드 프론트 공작 | 눈 트랙 담당 소녀 | 소녀 체력 강화 목적으로 트레이닝 커리큘럼 직접 작성. 소녀 거부. 황제 중재 |
| 도파민 팔라체 1위 가문 | 수도 트랙 소녀 | 라이벌 가문의 후원 소녀 정보 빼내려다 황립 보안국에 적발. 벌금 젤리 1만 개 |
| 사하라 리지 남작 | 사막 트랙 소녀 | 수분 공급 젤리 개발. 소녀들에게 인기 폭발. 다른 귀족들이 레시피 요구. 비공개 유지 중 |
| 클라우드 피크 백작 | 고산 트랙 소녀 | 독자 고산 통화 발행 시도. 적발 후 "이건 포인트 카드입니다"로 우겼으나 실패 |

### 6-3. 소녀들의 제국 적응기

| 상황 | 소녀들의 반응 |
|------|------------|
| 소환 직후 | "여기가 어디야?" → 황제 직접 환영 연설 10분 → "어… 알겠어요. 달릴게요" |
| 탈출 시도 | 포털 기술 독점을 뒤늦게 인지. "그럼 달리는 게 낫겠다" |
| 형무소 선택 | 입소 3일 후 "이거 나쁘지 않은데?" 6일 후 "레이스 신청 어떻게 해요?" |
| 포인트 만점 달성 | 황제 앞에서 소원 발표. 황제가 진지하게 수첩에 받아 적음 |
| 자유 시민권 선택 | 제국에 영주. 다음 시즌 "후배 소녀" 응원단 참여 |

### 6-4. 제국 외무부의 황당한 업무일지

> 제국 외무부 업무 비중 (공식 통계)

| 업무 | 비중 |
|------|------|
| 소녀 수급 외교 (타 세계 설득) | 55% |
| 포털 유지 협상 | 20% |
| 소녀 귀환 후 세계와 사후 처리 | 15% |
| 기타 전통 외교 | 10% |

- 외무부 장관 공식 직함: "제국 다중 세계 인재 수급 외교 총감독 겸 포털 협상 특명전권대사"
- 외무부 신입 연수 1주차 과목: "빠른 소녀를 설득하는 화법"
- 외무부 연간 보고서 1페이지: 금년 소환 소녀 명단 및 출신 세계 평가

### 6-5. 제국 헌법의 달리기 관련 조항

| 조항 | 원문 |
|------|------|
| 헌법 제1조 | "달리기는 신성하다. 황제가 친히 보증한다." |
| 헌법 제2조 | "제국 예산의 최우선은 국방이다. 단, 황제가 달리기 대회를 국방과 동등하다고 선언할 경우 예외로 한다." (현재 동등 선언 상태) |
| 헌법 제7조 | "귀족은 소녀 후원을 거부할 수 없다. 단 후원 소녀의 동의가 없을 경우 귀족이 직접 달린다." |
| 헌법 제12조 | "황제의 해설은 공식 역사 기록에 포함된다." |
| 헌법 제19조 | "도파민 젤리 위조는 반역죄에 준한다." |
