## 파트 1: 디자이너 가이드

### 1. 전달 폴더 구조

파일을 이 경로에 넣어주세요:

```
Assets/Design/Incoming/
└── [날짜_작업명]/          ← 오늘 날짜 + 작업 이름으로 폴더 생성
    ├─ 전달노트.md          ← 필수! 무엇을 어디에 쓰는지 설명
    ├─ 파일1.png
    ├─ 파일2.png
    └─ ...
```

예시:
```
Assets/Design/Incoming/
└── 20260302_배팅패널_아이콘교체/
    ├─ 전달노트.md
    ├─ character_icon_0.png
    ├─ character_icon_1.png
    └─ bet_type_win.png
```

### 2. 전달노트.md 작성 방법

`Assets/Design/Incoming/전달노트_양식.md` 파일을 복사해서 작성하세요.

**반드시 포함해야 할 내용:**

```markdown
## 작업 개요
[무슨 작업인지 한 줄 설명]

## 파일 목록

| 파일명 | 교체 대상 경로 | 비고 |
|--------|---------------|------|
| character_icon_0.png | Assets/Resources/Icon/Char/Character_0.png | 크기 64×64 |
| bet_type_win.png | Assets/Resources/Icon/bet_win.png | 신규 파일 |

## 레이아웃 변경 (있는 경우)
[없으면 "없음" 으로]

## 신규 오브젝트 (있는 경우)
[없으면 "없음" 으로]
```

### 3. 파일 명명 규칙

| 규칙 | 예시 |
|------|------|
| 영문 소문자 + 언더스코어 | `bg_desert.png` ✅ |
| 한글/공백/특수문자 금지 | `배경 사막.png` ❌ |
| 확장자 명시 | `.png`, `.ttf` 등 |

### 4. 이미지 Export 설정

| 항목 | 값 |
|------|-----|
| 포맷 | PNG (투명 배경 지원) |
| 색상 모드 | RGBA |
| 캐릭터 아이콘 크기 | 64×64 px |
| UI 아이콘 크기 | 32×32 또는 48×48 px |
| 배경 이미지 크기 | 1920×1080 px |
| 트랙 배경 크기 | 1920×1080 px |

> 픽셀아트 이미지는 **리사이즈하지 마세요** (Unity에서 Point Filter 설정으로 처리)

### 5. 레이아웃 변경이 있을 때

Unity 프리팹을 직접 수정한 경우, 수정 후 반드시 확인:
- Prefab Mode(`Prefabs/UI/BettingPanel.prefab` 더블클릭)에서 수정
- 오브젝트 이름은 절대 변경하지 말 것
- 수정 완료 후 전달노트에 변경 내역 기록

### 6. Git으로 제출

```bash
git add Assets/Design/Incoming/[날짜_작업명]/
git commit -m "[UI] 배팅패널 아이콘 교체 전달"
git push origin main
```

또는 **GitHub Desktop**:
- 변경된 파일 체크 → Summary 입력 → Commit → Push

## 파트 2: 개발자 가이드

### 1. 디자인 파일 받기

```bash
git pull origin main
```

`Assets/Design/Incoming/` 에 새 폴더가 생겼는지 확인.

### 2. Claude에게 적용 요청하기

Claude Code 실행 후 다음과 같이 입력:

```
Assets/Design/Incoming/[날짜_작업명]/전달노트.md 읽고 디자인 파일 적용해줘
```

**Claude가 하는 일:**
1. 전달노트 읽기
2. 각 파일을 올바른 Unity 폴더로 이동/복사
3. 교체 대상 파일 확인 및 덮어쓰기
4. 신규 파일인 경우 Import Settings 안내
5. 레이아웃 변경이 있으면 코드 수정 여부 확인

### 3. Unity에서 검증

1. Unity가 파일을 자동 임포트 (Console 에러 없는지 확인)
2. Import Settings 확인:
   - Filter Mode: **Point (no filter)**
   - Compression: **None**
3. Play 모드에서 해당 화면 확인

### 4. 적용 완료 후

```bash
# Incoming 폴더 정리 (Applied로 이동)
git add .
git commit -m "[C] [UI] 배팅패널 아이콘 교체 적용"
git push origin main
```

> Applied 폴더로 이동: `git mv Assets/Design/Incoming/[날짜_작업명] Assets/Design/Applied/[날짜_작업명]`


## 자주 쓰는 교체 경로 정리

| 에셋 종류 | Unity 경로 |
|-----------|-----------|
| 캐릭터 아이콘 (0~11) | `Assets/Resources/Icon/Char/Character_N.png` |
| 타이틀 로고 | `Assets/Resources/BG/main_title.png` |
| 타이틀 배경 | `Assets/Resources/BG/main_title_bg.png` |
| 트랙 배경 | `Assets/Resources/Tracks/bg_[이름].png` |
| UI 아이콘 | `Assets/Resources/Icon/[파일명].png` |
| 폰트 | `Assets/Fonts/[폰트명]/[파일명].ttf` |
| 캐릭터 일러스트 | `Assets/Resources/Char/[파일명].png` |
