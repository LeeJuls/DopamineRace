# UI 패널 배경 이미지 적용 가이드

**작성일**: 2026-06-20  
**참조 커밋**: `1b2d952` (ExchangeModal BG 적용), `cbbc6c3` (고양이 클릭 환전 팝업)

---

## 개요

이 프로젝트에서 팝업/패널 배경은 단색 Image가 아닌 **9-슬라이스 스프라이트**를 사용한다.
BettingPanel, ExchangeModal 등이 이 패턴을 공유한다.

```
패널 GameObject
  └── Image (m_Sprite = Img_BG_ListSlot_BG_01, type = Sliced, color = white)
        └── ... 자식 UI 요소들 (TitleText, Button, ...)
```

---

## 배경 이미지 에셋

| 파일 | 크기 | 용도 |
|------|------|------|
| `Assets/Resources/UI/Img_BG_ListSlot_BG_01.png` | 112×112px | 팝업 패널 기본 배경 |
| `Assets/Resources/UI/Img_BG_ListSlot_BG_02.png` | 동일 | 강조/선택 패널 |
| `Assets/Resources/UI/Img_BG_ListSlot_BG_03.png` | 동일 | 비활성 패널 |
| `Assets/Resources/UI/Img_BG_ListSlot_BG_04.png` | 동일 | 특수 패널 |

> **주의**: border 2px (Sprite Editor에서 확인). sub-sprite가 있는 경우 border 오버라이드가
> `{0,0,0,0}`이 되어 Sliced 효과가 무효화될 수 있음 → 그 경우 `Image.Type.Simple` 사용.

## 프리팹 크리에이터에 배경 이미지 적용하는 방법

### 1. 적용 위치

프리팹 크리에이터(`Assets/Scripts/Editor/` 또는 `Assets/Editor/`)의  
`Build*Prefab()` 함수에서 패널 GameObject에 Image를 추가하는 부분을 수정한다.

### 2. 코드 패턴

```csharp
// ❌ 단색 방식 (구버전)
panel.AddComponent<Image>().color = new Color(0.12f, 0.10f, 0.18f, 0.95f);

// ✅ 배경 이미지 방식 (현재 표준)
Sprite bgSprite = AssetDatabase.LoadAssetAtPath<Sprite>(
    "Assets/Resources/UI/Img_BG_ListSlot_BG_01.png");
Image panelImg = panel.AddComponent<Image>();
panelImg.raycastTarget = false;          // 패널 자체는 raycast 불필요
if (bgSprite != null)
{
    panelImg.sprite     = bgSprite;
    panelImg.type       = Image.Type.Sliced;
    panelImg.fillCenter = true;
    panelImg.color      = Color.white;
}
else
{
    Debug.LogWarning("[패널명] Img_BG_ListSlot_BG_01 없음 — 단색 폴백");
    panelImg.color = new Color(0.12f, 0.10f, 0.18f, 0.95f);
}
```

### 3. 핵심 주의사항

| 항목 | 이유 |
|------|------|
| `AssetDatabase.LoadAssetAtPath` 사용 | Editor 스크립트 전용. `Resources.Load`는 런타임 API라 Editor에서 불안정 |
| 변수명 `panel` 유지 | `modal`로 혼용 시 NRE 발생. 기존 변수명 확인 필수 |
| `raycastTarget = false` 명시 | 패널 배경이 자식 버튼의 클릭 이벤트를 가리는 현상 방지 |
| `Debug.LogWarning` 추가 | 에셋 누락 시 단색 폴백 — 콘솔에서 즉시 감지 가능 |

## 적용 절차

```
1. 해당 크리에이터 스크립트 수정 (위 코드 패턴 적용)
2. Unity 컴파일 완료 확인 (콘솔 에러 0)
3. DopamineRace → 프리팹 생성 → [해당 프리팹] 메뉴 실행
4. Inspector 확인: 패널 Image.sprite 필드에 BG_01 할당 여부
5. Play → 팝업 열기 → 시각 확인
```

## 검증 기준

| 항목 | Pass 조건 |
|------|----------|
| 팝업 배경 텍스처 표시 | 단색이 아닌 BG_01 이미지 표시 |
| Inspector Image.sprite | `Img_BG_ListSlot_BG_01` 표시 (null 금지) |
| 콘솔 Warning | `"없음 — 단색 폴백"` 문자열 없음 |
| 자식 버튼 클릭 | 배경 Image에 가려지지 않고 정상 동작 |
