# 경로 가이드 (참조용 — 디자이너용)

## 프리팹 목록

| 프리팹 | 포함 버튼/UI |
|--------|-------------|
| **BettingPanel** | 배팅 버튼, 시작 버튼, 탭 버튼, 캐릭터 선택 |
| **ResultPanel** | 다음 라운드 버튼, 순위 배지, 순위 화살표 |
| **FinishPanel** | 새 게임 버튼, Top100 버튼 |
| **LeaderboardPanel** | 닫기 버튼 |
| **CharacterItem** | 캐릭터 아이콘, 컨디션 아이콘, 정보 표시 |

## 자주 교체하는 파일 경로

| 항목 | Unity 경로 | 크기 |
|------|-----------|------|
| 캐릭터 아이콘 N번 | `Assets/Resources/Icon/Char/Character_N.png` | 64x64 |
| 컨디션 아이콘 | `Assets/Resources/Icon/condition_[상태].png` | 24x24 |
| 탭 버튼 기본/선택 | `Assets/Resources/UI/btn_tab_normal/selected.png` | 가변 |
| 시작 버튼 기본/비활성 | `Assets/Resources/UI/btn_start_normal/disable.png` | 가변 |
| 패널 배경 | `Assets/Resources/UI/panel_bg.png` | 가변 |
| 타이틀 로고 | `Assets/Resources/BG/main_title.png` | 픽셀아트 |
| 타이틀 배경 | `Assets/Resources/BG/main_title_bg.png` | 1920x1080 |
| 트랙 배경 | `Assets/Resources/Tracks/bg_[이름].png` | 1920x1080 |
| 캐릭터 일러스트 | `Assets/Resources/Char/[파일명].png` | 자유 |
| 폰트 | `Assets/Fonts/[폰트명]/[파일명].ttf` | - |

## 파일 명명 규칙

- 영문 소문자 + 언더스코어: `btn_start_normal.png` O
- 한글/공백/특수문자 금지: `시작 버튼.png` X

## Import 설정 (개발자 참조)

- **Filter Mode**: Point (no filter) — 픽셀아트 필수
- **Compression**: None
- **Texture Type**: Sprite (2D and UI)
