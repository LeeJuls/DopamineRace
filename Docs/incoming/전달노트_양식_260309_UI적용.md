> **Button 상태**: Normal / Highlighted / Pressed / Disabled
> **Toggle 상태**: (ON) / (OFF) — Button 용어(Pressed 등) 사용 금지
> **선택형 탭 버튼**: `(비선택 4상태)` / `(선택 4상태)` 표기 + 비고에 `Normal/Highlighted/Pressed/Disabled 순` 기재 — 코드 수정 여부는 개발자가 판단 (디자이너는 몰라도 됨)
> **공용 스프라이트**: 여러 오브젝트가 같은 이미지를 쓰면 비고에 `N탭 공용` 기재 — 예: `6탭 공용`
> **자식 오브젝트**: `부모/자식` 형식으로 — 예: `HideInfoToggle/CheckBg (OFF)`
> **신규 파일**: 비고에 `신규 추가` 표기
> **Unity 기본 유지**: 파일 없이 오브젝트 이름 + 비고에 Color(HEX) 기재
> **Color 초기화 필요 시**: 비고에 `Color 흰색으로 초기화` — 기존 틴트가 걸린 오브젝트에 스프라이트 교체할 때 사용
