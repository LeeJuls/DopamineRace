### 3-2. 소형 캡슐 (231×87px) 개선안 — 최우선

**핵심 원칙: "캐릭터가 로고보다 커야 한다"**

```
[개선안 레이아웃]
┌─────────────────────────────┐
│  [도파니 캐릭터]  ========> │  ← 속도선 (오른쪽 방향)
│  달리는 자세,    D★pamine   │
│  상반신 이상     Racing(소)  │
└─────────────────────────────┘
```

**필수 요소**:
- **캐릭터**: 인기 도파니 1~2명, 달리는 측면/¾뷰 포즈, 전체 높이의 70% 이상
- **표정**: 결의/흥분 — 눈이 반드시 보여야 함
- **속도선**: 캐릭터 뒤에서 수평으로 뻗어나가는 모션라인
- **로고**: 우측 상단, 전체 면적의 20% 이하
- **배경**: 트랙 원근감 또는 단색+그라디언트 (산/숲 배경 제거)
- **색감**: 현재 녹색/주황 에너지 유지, 대비 강화

**금지 사항**:
- 로고만 있는 구성
- 캐릭터 전신이 소형에서 1cm 미만
- 어두운 배경 (Steam 검색목록 자체가 어두움 → 묻힘)

## 4. About This Game — GIF 삽입 가이드

Steam 설명란에 GIF/WEBM 삽입 시 CTR→체류시간→위시리스트 전환율 향상 효과 있음.

### 4-1. 삽입 방법

```
Steamworks > 설명 탭 > 게임 정보 에디터
→ 이미지 버튼(🖼) 클릭
→ 오른쪽 "맞춤 이미지 업로드" 섹션에 파일 업로드
→ 에디터 본문에서 이미지 위치 지정

지원 형식: GIF, WEBM(vp9, 투명도 지원), APNG
권장 가로: 780px 이하 (데스크탑 기준)
권장 용량: 5MB 이하 (로딩 속도)
루프 길이: 2~5초 권장
```

### 4-2. 추천 GIF 3종 (우선순위 순)

---

#### 🥇 GIF ① — 결승선 역전 장면 (최우선 제작)

```
위치: 게임 정보 최상단 (훅 텍스트 바로 아래, 현재 타이틀 이미지 대체 또는 병행)
길이: 2~3초 루프
```

**내용**:
- 3등 도파니가 마지막 직선주로에서 1, 2등을 추월
- 결승선 통과 순간 + 순위 UI가 바뀌는 장면 포함
- 관중석 반응 또는 팡파레 효과 있으면 베스트

**왜 이게 최우선인가**:
> 글을 읽지 않아도 "이 게임이 왜 쫄깃한가"를 즉시 전달.
> 역전 드라마가 게임의 핵심 감정인데, 현재 텍스트로만 설명되고 있음.

**편집 가이드**:
```
소스: 게임 내 가장 극적인 역전 레이스 클립
편집: 결승선 직전 1~2초 + 결과 확정 0.5초
속도: 약간 빠르게 (1.2~1.5x) → 긴박감 강화
텍스트 오버레이: 없어도 됨 (장면 자체가 설명)
```

```

**내용**:

> 텍스트 3줄 읽는 것보다 5초 GIF 한 번이 더 효과적.

**편집 가이드**:
```
3단계를 빠른 컷 전환으로:
[배팅 UI 1초] → [레이스 2초] → [결과 1초]
배당률 숫자와 캐릭터 이름이 보여야 "이 게임에 뭔가 있다"는 느낌
