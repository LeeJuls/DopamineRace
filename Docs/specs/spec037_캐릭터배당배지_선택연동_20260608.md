# SPEC-037 — 캐릭터 배당 배지 선택 연동 미리보기

**날짜**: 2026-06-08
**상태**: 구현 완료
**관련 히스토리**: `Docs/history/20260608_배팅배지선택연동_리더보드타이틀수정_히스토리.md`

## 배경
배팅 화면 캐릭터 리스트의 배당 배지(예 "1.5x")가 **단승 배당 고정**이라, 베팅 종류 탭(단승/연승/복승/쌍승/삼복승/복연승)과 캐릭터 선택 상태에 무관하게 표시됐다. 사용자가 "이 캐릭터를 고르면 배당이 얼마가 되는지"를 캐릭터별 배지로 미리 보고 싶다고 요구.

## 요구사항
- **R1** 배지 = "이 캐릭터를 클릭하면 되는 조합 배당" 미리보기.
- **R2** 복승 이상 0명 선택 시 각 배지 = 첫 배율 **1.1** (조합 미완성).
- **R3** 미선택 + 미완성: `현재 selections + 이 캐릭터` 조합값.
- **R4** 미선택 + 완성(IsComplete): `[이 캐릭터]` 단독 (클릭 시 리셋+추가 = 교체와 일치).
- **R5** 이미 선택됨: 현재 selections 조합값.
- **R6** 캐릭터 선택/해제·탭 전환·라운드 진입 시 즉시 갱신.
- **R7** 단승/연승(1명)은 기존 동작 유지. 보상 계산(`GetExpectedOdds`/`CalcPayout`)·점수 로직 불변.
- **R8** 신규 UI 문자열 없음(숫자+"x") — StringTable 변경 불필요.

## 설계

### 배지 산식 — `OddsCalculator.GetPreviewOdds(bet, racerIndex, racers)`
캐릭터 X 상태별 가상 selections로 `CalcComboOdds(betType, winOdds들)` 계산:

| X 상태 | 가상 selections | 처리 |
|--------|-----------------|------|
| 이미 선택됨 | 현재 selections (참조, 읽기 전용) | 현재 조합 |
| 미선택 + 미완성 | selections **복사** + X | 추가 미리보기 |
| 미선택 + 완성 | `[X]` 단독 | 교체 미리보기 |

→ 원본 `bet.selections` 미변경(미선택 분기는 복사). `CalcComboOdds`·`GetWinOdds` 재사용.

### 갱신 트리거
| 트리거 | 경로 |
|--------|------|
| 선택/해제 | `OnRacerClicked` → `UpdateButtonVisuals` → `RefreshOddsBadges` |
| 탭 전환 | `OnBetTypeClicked` → `UpdateButtonVisuals` → `RefreshOddsBadges` (selections 리셋 후 0명 기준) |
| 라운드 진입 | `ResetBetting` → `RefreshCharacterItems` / `UpdateButtonVisuals` |

→ `UpdateButtonVisuals` 끝에 `RefreshOddsBadges()` 통합해 3경로 일원화.

## 비고
- 복승 이상에서 약체 조합은 상한(`oddsMax_*`)에 포화될 수 있음 — 의도된 게임 동작(강체=저배당, 약체=고배당).
