**작성일**: 2026-03-01
**대상**: UI 디자이너

## 개요

배팅 화면 전체를 픽셀아트 스타일로 디자인하기 위한 리소스 목록 및 작업 범위 정의.

## 2. 버튼

### 2-1. 탭 버튼 (배팅 타입 선택)

현재 6종: 단승 / 연승 / 복승 / 삼승 / 상복승 / 복연승

| 상태 | 설명 |
|------|------|
| Normal | 기본 (비활성 탭) |
| Selected | 선택된 탭 (활성) |
| Over | 마우스 오버 |
| Disable | 비활성화 (해당 타입 불가) |

**전달 방식**: 9-slice 가능한 단일 PNG 또는 상태별 PNG
**적용 경로**: `Assets/Resources/UI/btn_tab_[상태].png`

> 현재 탭은 코드로 색상만 변경 중. 커스텀 이미지로 교체 예정.

### 2-2. 경기 시작 버튼 (Start)

| 상태 | 설명 |
|------|------|
| Normal | 기본 (초록 계열 권장) |
| Over | 마우스 오버 |
| Press | 클릭 시 |
| Disable | 배팅 선택 전 (회색) |

**적용 경로**: `Assets/Resources/UI/btn_start_[상태].png`

### 2-3. 리스트 아이템 배경 (캐릭터 행)

| 상태 | 설명 |
|------|------|
| Normal | 기본 행 배경 |
| Selected | 캐릭터 선택 시 |
| Over | 마우스 오버 |

**적용 경로**: `Assets/Resources/UI/list_item_[상태].png`

## 3. 정보 영역 (텍스트 위주)

> **작업 방향**: 스트링 위주, 개별 아이콘/이미지 디자인 최소화.
> 레이아웃 그룹핑과 배경(BG) 위주로 작업.

| 항목 | 내용 | 비고 |
|------|------|------|
| 배팅 하단 정보바 | `1라운드 배팅 선택! | 단승 석 포인트 | 배팅: 1 → 27` | 레이아웃 + BG |
| 트랙 정보 토글 패널 | 트랙명, 거리, 타입, 트랙 설명 텍스트 | 텍스트 + 프레임만 |
| 캐릭터 경기 기록 | 최근 경기 순위 텍스트 (`1st.5%, 1st.8%` 등) | 텍스트 전용 |

## 4. 컨디션 아이콘 교체

현재 화살표(↑↑ ↑ → ↓ ↓↓) 심볼을 **새로운 디자인 심볼로 교체** 예정.

| 컨디션 | 현재 | 교체 방향 | 권장 색상 | Unity 경로 |
|--------|------|-----------|-----------|-----------|
| 최상 | ↑↑ | 새 심볼 | `#44FF44` 초록 | `Assets/Resources/Icon/condition_best.png` |
| 상 | ↑ | 새 심볼 | `#AAFF66` 연초록 | `Assets/Resources/Icon/condition_good.png` |
| 중 | → | 새 심볼 | `#FFFFFF` 흰색 | `Assets/Resources/Icon/condition_normal.png` |
| 하 | ↓ | 새 심볼 | `#FFAA44` 주황 | `Assets/Resources/Icon/condition_bad.png` |
| 최하 | ↓↓ | 새 심볼 | `#FF4444` 빨강 | `Assets/Resources/Icon/condition_worst.png` |

**크기**: 24×24 px
**포맷**: PNG (투명 배경)

> 현재는 코드(`ConditionIconFactory.cs`)에서 런타임으로 화살표를 생성 중.
> 이미지로 교체 시 해당 코드 비활성화 처리 필요 — 개발자에게 알릴 것.
