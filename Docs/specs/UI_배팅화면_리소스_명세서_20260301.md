# UI 배팅화면 리소스 명세서

**작성일**: 2026-03-01
**대상**: UI 디자이너
**상태**: 작업 계획 (디자이너 배정 전)

---

## 작업 범위 (가능 / 불가)

### ✅ 가능
- 메인화면 UI 시안
- 확정 시 정해진 사이즈로 리소스화 및 분리 (PNG)

### ❌ 불가 (개발자 또는 향후 별도 처리)
- 엔진 작업 및 프리팹 / 테스트
- UI 이펙트 (머터리얼, 플러그인 등)
- UI 연출 (버튼·위젯 애니, 팝업 트랜지션 등)
- 이미지 폰트 작업
- 로고 및 타이틀
- UX 검토
- 캐릭터
- **아이콘 전반** (캐릭터 아이콘, 컨디션 아이콘 등) — 시간 부족으로 제외

> **아이콘 관련 처리 방침**
> - 캐릭터 아이콘 (12종): 현재 플레이스홀더 유지, 별도 일정 필요
> - 컨디션 아이콘: 코드 생성(`ConditionIconFactory.cs`) 그대로 유지

## 2. 버튼

### 2-1. 탭 버튼 (배팅 타입 선택)

6종 탭: 단승 / 연승 / 복승 / 삼승 / 상복승 / 복연승

| 상태 | 설명 |
|------|------|
| Normal | 기본 (비활성 탭) |
| Selected | 선택된 탭 (활성) |

**전달 방식**: 상태별 PNG 2종 (Normal / Selected)
**크기**: 가변 (9-slice 권장)
**Unity 경로**: `Assets/Resources/UI/btn_tab_normal.png`, `btn_tab_selected.png`

> 현재 코드로 색상만 변경 중 → 이미지로 교체 예정.
> Over / Press 상태는 별도 협의 후 결정.

### 2-2. 경기 시작 버튼 (Start)

| 상태 | 설명 |
|------|------|
| Normal | 기본 |
| Disable | 배팅 선택 전 (회색) |

**전달 방식**: 상태별 PNG 2종
**Unity 경로**: `Assets/Resources/UI/btn_start_normal.png`, `btn_start_disable.png`

### 2-3. 리스트 아이템 배경 (캐릭터 행)

| 상태 | 설명 |
|------|------|
| Normal | 기본 행 배경 |
| Selected | 캐릭터 선택 시 |

**전달 방식**: 상태별 PNG 2종
**Unity 경로**: `Assets/Resources/UI/list_item_normal.png`, `list_item_selected.png`

## 3. 정보 영역 (텍스트 위주, 레이아웃 BG만)

> **작업 방향**: 스트링 위주, 개별 이미지 최소화. 레이아웃 그룹핑과 배경(BG)만.

| 항목 | 내용 | 비고 |
|------|------|------|
| 배팅 하단 정보바 | `1라운드 배팅 선택! \| 단승 석 포인트 \| 배팅: 1 → 27` | BG + 레이아웃만 |
| 트랙 정보 토글 패널 | 트랙명, 거리, 타입, 설명 텍스트 | 프레임 BG만 |
| 캐릭터 경기 기록 | 최근 경기 순위 텍스트 | 텍스트 전용, 이미지 없음 |
