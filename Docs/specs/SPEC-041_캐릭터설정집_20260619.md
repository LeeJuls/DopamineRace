## 000 침묵 (goyo)

| 항목 | 내용 |
|------|------|
| 출신 세계 | 동방 왕국 〈영화국〉 야인 계열 |
| 레이스 소울 | 불꽃(焰) |
| 타입 | Runner |
| 나이 | 17세 |
| 성격 키워드 | 무언(無言), 절대적 확신, 냉정한 완벽주의 |
| 역할 | 영화국 야인 — 이름 없는 최강자 |
| 동기 패턴 | 증명 욕구형 |

**한 줄 소개:** 말을 아끼는 것이 아니라, 달리는 것으로 모든 말을 대신하는 소녀.

**배경 스토리:** 영화국에서 침묵은 이름도 문파도 없는 야인이었다. 도전자들이 끊임없이 찾아왔고, 그녀는 매번 이겼다. 인정도 칭찬도 없었다. 강하기 때문에 외로웠고, 외롭기 때문에 더 강해졌다. 소환되는 순간에도 그녀는 책을 읽고 있었다 — 빗소리를 들으며. 아레나에서도 자신을 증명하는 방식은 변하지 않는다. 말 없이, 먼저 결승선에 닿는 것.

**스킬 내러티브:** 속도·가속·지능 모두 최고 수준(Spd20/Acc20/Int20). "항상 속도 +6%" 패시브는 감정 기복 없이 일정하게 최고를 유지하는 그녀의 성격 그 자체다. 특별한 때를 노리지 않는다. 처음부터 끝까지 전력이 곧 전략이다.

**주요 관계:**
- **라이벌 — 황금호**: "처음과 마지막". 침묵은 앞을 보고, 황금호는 그 등을 본다. 두 사람은 레이스 내내 얼굴을 마주치지 않는다
- **라이벌 — 홍염**: 같은 동방 왕국 출신 도주형. 침묵은 홍염이 유일하게 진심으로 싸운다고 느끼지만 절대 말로 표현하지 않는다

|------|------|
| 출신 세계 | 동방 왕국 〈영화국〉 야인 계열 |

| 타입 | Runner |
| 나이 | 16세 |
| 성격 키워드 | 뜨거운 투지, 감정 과잉, 스퍼트의 화신 |
| 역할 | 영화국 야인 — 불꽃을 다루는 소녀 |
| 동기 패턴 | 증명 욕구형 |

**한 줄 소개:** 침묵의 등을 한 번도 앞서지 못했지만, 한 번도 포기하지 않은 소녀.

**배경 스토리:** 홍염은 영화국 전체에서 침묵 다음으로 빠른 소녀였다. "다음으로 빠르다"는 말이 항상 따라붙었고, 그 말이 싫었다. 그녀의 화는 레이스에서 폭발로 변했다 — 특히 마지막 구간. 소환된 아레나에서도 침묵은 여전히 앞에 있었다. 홍염은 그것이 오히려 기뻤다. 이길 상대가 여기서도 있다는 뜻이었으니까.

**스킬 내러티브:** 스퍼트 구간 속도 +4% 패시브는 마지막 100m에 모든 것을 쏟는 성격의 반영이다. 전반에 에너지를 조금 아끼고, 막판에 불꽃처럼 타오른다. 스태미나(Sta19)가 탄탄해 소모를 버틸 수 있다.

**주요 관계:**
- **라이벌 — 침묵**: 홍염만이 침묵의 침묵이 무엇을 의미하는지 안다. 하지만 먼저 말을 걸지는 않는다

## 003 크리스탈 (crystal)

| 항목 | 내용 |
|------|------|
| 출신 세계 | 크로노스 시티 |
| 레이스 소울 | 불꽃(焰) |
| 타입 | Runner |
| 나이 | 17세 |
| 직업 | 해커 |
| 동기 패턴 | 소원 추구형 |

**한 줄 소개:** 레이스를 데이터로 분석하고, 충돌을 계산해 승리를 가져가는 소녀.

**배경 스토리:** 크로노스 시티에서 크리스탈은 도시 보안 시스템을 뚫고 최단 경로를 찾는 해커였다. 레이스는 그녀에게 코드와 같았다 — 변수를 분석하고, 충돌 포인트를 계산하고, 승리를 설계한다. 몸이 약한 것(Pow5)은 알지만, 머리가 있으면 충분하다. 그녀가 아레나에서 원하는 것은 아직 말하지 않았다.

**스킬 내러티브:** 충돌 5회 누적 → 5초 충돌 필승 액티브 스킬(쿨15s). 충돌 자체를 의도적으로 유도하는 전략적 도주 스타일. 지능(Int20)과 운(Lck20)으로 계산하고, 최저 힘(Pow5)이라는 약점을 역이용한다.

**주요 관계:**
- **공생 — 세크레트**: 서로의 약점을 아는 정보 거래 구도. 크로노스 시티에서도 알던 사이라는 소문이 있다

**한 줄 소개:** 서있는 자를 이겨야 안심이 되는 소녀. 1등 외에는 의미가 없다.

**배경 스토리:** 아마존은 황금 기사단 소속이 아니었다. 홀로 성장해 기사단원들을 하나씩 이기며 자신을 증명했다. 그러나 공식 서열은 결코 인정받지 못했다 — 기사단은 이미 해체됐고, 정식 대결은 없었다. 아레나야말로 진정한 실력으로만 판가름 나는 곳이라고 믿는다.

**스킬 내러티브:** 충돌 4회 누적 → 5초 필승(쿨20s). 충돌을 빠르게 쌓는 공격적 도주 스타일. 속도·가속 최고(Spd20/Acc20)지만 스태미나(Sta11)가 낮아 초중반 집중 승부를 노린다.

**주요 관계:**
- **앙숙 — 정의**: 같은 기사단 출신 전사지만 신념이 충돌. 아마존은 "규칙이 강자를 가두는 도구"라고 생각한다

**주요 관계:**
- **공생 — 크리스탈**: 서로의 비밀을 아는 유일한 상대. 레이스 밖에서의 관계가 레이스에도 영향을 준다

| 나이 | 22세 |

| 역할 | 황금 기사단 전 단장 |
| 동기 패턴 | 증명 욕구형 |

**한 줄 소개:** 기사단은 사라졌지만, 그 황금빛 의지는 아직 달리고 있다.

**배경 스토리:** 황금 기사단을 이끌었던 단장. 신조 충돌로 기사단이 해체될 때, 골든데이는 자신이 옳다고 믿었지만 그것이 분열의 원인이 됐다고도 안다. "질서 속 승리만이 진정한 승리"라는 신조를 지키며 아레나에서도 정석을 고집한다. 임팩트와 정의가 눈앞에 있을 때도 흔들리지 않는 것이 목표다.

**스킬 내러티브:** 피격 충돌 3회 → 6초 필승(쿨18s) 액티브. 맞으면서도 버티고, 역경을 쌓아 폭발시키는 스타일. 균형 잡힌 최고 스탯(Spd20/Sta18)이 오래 선두를 지탱한다.

**주요 관계:**
- **내부 갈등 — 임팩트, 정의**: 기사단 삼각 구도. 서로를 가장 잘 알기에 레이스에서 가장 치열하게 견제한다
- **동료 — 전마**: 기사단이 해체된 후에도 옆에 있는 존재

- **배신·신뢰 — 동풍**: 레이스 페이스메이커 관계. 어느 날 결정적 순간에 동풍이 먼저 치고 나갔다

**주요 관계:**

- **배신·신뢰 — 나리**: 암묵의 협력 관계가 깨진 날 이후로도 두 사람은 계속 나란히 달린다

**배경 스토리:** 바람 네 자매와 달리, 독수리는 하나의 완전한 정령이다. 아레나 상공을 떠돌다 레이스에 참가하기 시작했다. 이유는 있다 — "더 높이 날기 위한" 무언가를 원한다. 아직 그것이 무엇인지 구체적이지 않지만, 레이스를 반복하면서 조금씩 선명해지고 있다.

**스킬 내러티브:** 스퍼트 슬립스트림 150% 패시브. 후반 스퍼트에서 상대의 기류를 타고 급습한다. 균형 잡힌 스탯(Spd17/Sta16)으로 안정적으로 후반을 노린다.

**주요 관계:**
- **공명 — 북산**: 산과 하늘. 정령 계열이지만 바람 사자매와는 다른 결의 존재

**주요 관계:**
- **공감 — 레이디**: 하위 순위에서 회복하는 같은 스타일. 두 사람은 레이스 후 말 없이 공감한다

**한 줄 소개:** 가을 서풍처럼 냉혹하다. 선두권이 보이면 그 에너지를 빼앗는다.

**배경 스토리:** 바람 정령의 가을·서쪽 분열체. 동풍이 봄을 알린다면, 서풍은 모든 것이 지는 계절을 담당한다. 선두권에 들어가면 상대의 에너지를 드레인하는 독특한 패턴 — 강한 것이 있으면 그것을 가져간다. 하나로 돌아가고 싶지 않다. 지금 이 냉혹함이 자신의 전부이기 때문에.

**스킬 내러티브:** 상위 6등 내 드레인 70% 패시브. 선두권에 들어오는 순간부터 주변 주자의 에너지를 가져간다. 힘(Pow20)이 최고 수준으로 드레인 강도가 높다.

**주요 관계:**
- **정령 관계 — 동풍·광풍·천리풍**: 사분된 하나. 서풍은 합체에 가장 무감하다 — 어차피 모든 것은 지는 것이라는 시각
- **대비 — 동풍**: 봄과 가을. 가장 먼 분열체

## 031 천리풍 (cheonripung)

| 항목 | 내용 |

| 출신 세계 | 자연 정령계 |
| 레이스 소울 | 대지(地) |
| 타입 | Reckoner |

| 역할 | 바람 정령 — 겨울·북쪽 분열체 |
| 동기 패턴 | 소원 추구형 |

**한 줄 소개:** 천 리를 달리는 바람. 네 조각 중 가장 빠르고, 그래서 가장 외롭다.

**배경 스토리:** 바람 정령의 겨울·북쪽 분열체. 네 조각 중 가장 빠르다. 동풍, 광풍, 서풍이 각자의 계절을 담당한다면, 천리풍은 "끝없는 거리"를 담당한다 — 방향이 아니라 거리. 하나로 돌아가는 것이 소원이다. 가장 빠르게 달리면, 언젠가 나머지 셋을 따라잡을 수 있을 것 같은 느낌이 든다.

**스킬 내러티브:** 추격 충돌 4회 → 6초 필승(쿨22s) 액티브. 가속(Acc20) 최고로 순간 위치 이동이 빠르다. 스태미나(Sta18)도 높아 후반 추격을 오래 유지할 수 있다.

**주요 관계:**
- **정령 관계 — 동풍·광풍·서풍**: 사분된 하나. 천리풍은 유일하게 합체를 원한다. 그래서 나머지 셋과 레이스에서 가장 많이 충돌한다
