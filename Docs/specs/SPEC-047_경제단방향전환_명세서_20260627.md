# SPEC-047 경제 단방향 전환 — 젤리 소멸·스톤만 획득

> 2026-06-27 · SPEC-028/035 개정 · 적중 시 젤리 미반환, 스톤만 보상

## 1. 배경

| 항목 | 내용 |
|------|------|
| 문제 | 적중 시 젤리·스톤을 둘 다 `ceil(betAmount×배당)` 지급 → 젤리 복리 폭증, 게임오버/구제/고양이의힘 무의미 |
| 원인 | SPEC-035(2026-05-17)에서 "직관성"을 위해 `stone=jelly=ceil(betAmount×배당)`로 변경한 결정 |
| 조치 | SPEC-035를 의도적으로 되돌려 **단방향 경제**로 개정 |

## 2. 개정 경제 규칙

| 시점 | 동작 |
|------|------|
| 배팅(TryBet) | 젤리에서 베팅액만큼 차감 (변경 없음) |
| 적중 | **스톤만** 획득 = `ceil(betAmount × 배당)`. 젤리는 반환하지 않음 |
| 미적중 | 보상 없음. 젤리는 배팅 시 차감된 것 외 추가 차감 없음 |

- 예) 젤리 100 중 20 베팅 → 즉시 80. 적중=스톤 +`ceil(20×배당)`(젤리 80 유지), 미적중=스톤 0(젤리 80 유지)
- 젤리 = 한정 "연료" / 스톤 = 누적 성과 + 고양이의힘으로 젤리 환전
- **밸런스(catPowerUsesPerGame 등)는 본 SPEC 범위 외** — 수치 변경·백테스트 없음

## 4. 도전과제 조정 (SPEC-046 연동)

| API Name | 기존 | 개정 |
|----------|------|------|
| ACH_BIG_WIN | 한 라운드 젤리 500+ | **한 라운드 스톤 500+** |
| ACH_RICH | 젤리 1000+ 보유 | **누적 스톤 3000+ 보유** (스톤이 더 빨리 쌓여 재조정) |

> RICH_STONE_THRESHOLD는 상수라 오너가 추후 조정 가능. 파트너 사이트 이름/설명은 SPEC-046 §3 갱신본 참조.
