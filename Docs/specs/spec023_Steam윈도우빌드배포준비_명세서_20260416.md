## 빌드 절차

1. Unity에서 Windows x64 IL2CPP 빌드 → `D:\Builds\DopamineRace_v0.1.0\`
2. 빌드 결과물을 `C:\steamcmd\content\DopamineRace\`로 복사
3. SteamCMD 업로드:
   ```
   C:\steamcmd\steamcmd.exe +login clauzbt +run_app_build C:\steamcmd\scripts\app_build_4532310.vdf +quit
   ```
4. Steamworks 대시보드에서 빌드 확인 → Publish

## IL2CPP 설치 이슈 해결 기록

- **문제**: Unity Hub에서 IL2CPP 설치 시 "Validation Failed"
- **원인**: 바이러스 백신(AhnLab) 스캔으로 인한 120초 Timeout
- **해결**: `%APPDATA%\UnityHub\downloads\UnitySetup-Windows-IL2CPP-Support-for-Editor-6000.3.7f1.exe` 직접 관리자 권한 실행
- **확인**: `C:\Program Files\Unity\Hub\Editor\6000.3.7f1\Editor\Data\il2cpp\` 폴더 존재 확인

## Unity PackageManager 에러 (미해결)

- **증상**: Unity 에디터 시작 시 "Failed to start Unity Package Manager local server process"
- **원인**: `HOME` 환경변수 미설정 → UPM(Node.js) `getLocalConfigFolder` 크래시
- **부분 조치**: `HOME = C:\Users\USER` 사용자 환경변수 등록
- **현재 상태**: Unity Hub를 통해 열어야 정상 동작 (Hub가 환경변수 셋업)
- **다음 단계**: Unity Hub → Projects → DopamineRace 로 열기
