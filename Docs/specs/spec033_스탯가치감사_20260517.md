# SPEC-033 — 스탯-가치 감사 + 숨은 보정 레이어(휴면) + 3그룹 공정성 진단

**날짜**: 2026-05-17
**상태**: ✅ 진단 완료 (숨은 레이어 인프라 구축·휴면 / 재보정 불필요 결론)
**선행**: SPEC-031(지능 종결)·SPEC-032(타입 매크로 σ≤5 동결)
**방식**: Harness — 통제 단일스탯 스윕 계측 + 실스탯 코호트 감사

## 배경
SPEC-031/032 이후 스탯 간 1포인트당 레이스 가치 드리프트 의심. 개별 캐릭터
스탯은 의도적 격차라 불변, 글로벌 스탯-가치만 점검.

## 구축물 (보존·휴면)
- **통제 단일스탯 스윕 계측**: `RaceBacktestWindow.cs` `statSweepMode` — 8레이서
  동일 가상캐릭, 슬롯별 1스탯만 차등, 값버킷 로테이션(위치편향 상쇄), 시드 고정.
  G0 게이트 PASS(χ²=10.40<14.07, 시드 재현성 동일β).
- **숨은 보정 레이어**: `Assets/Resources/Data/HiddenStatWeights.csv`(6값) +
  `HiddenStatWeights.cs` 로더. 실계산 `effective = 표시값 × hidden_<stat>`,
  유저는 1~20 표시 유지. `RacerController_V4`(8배선)+`RaceBacktestWindow`(7미러).
  **현재 전부 1.0 = 무변화 no-op**(β·σ 원본 완전 일치 검증). 미래 레버로 휴면 보존.

## G1 측정 — 스탯당 승률가치 β (hidden=1.0, %p/+1pt, SPD=1.00 정규화)
| SPD | STA | INT | POW | LUCK | ACC |
|-----|-----|-----|-----|------|-----|
| 6.01(1.00) | 3.54(0.59) | 2.77(0.46, 타입의존 R3.9~C1.5) | 1.83(0.30) | 1.83(0.30) | 1.45(0.24) |
→ **SPD가 타 스탯 ~4배 지배**(드리프트 실재). 3그룹: Core(SPD,STA)/Support(ACC,POW)/Variable(INT,LUCK).

## 튜닝 시도 → 기각 (정직 기록)
hidden_speed 감쇠로 Core 등동화(SPD↔STA) 시도 → **SPEC-032 타입 σ 회귀**
(1.98~3.81 → 7.22) + 고요(SPD20·INT20·STA18 다중-max)는 SPD만 낮춰도 59~77% 잔존.
타입 부스트/스퍼트 경제가 옛 SPD가치 기준 튜닝이라 SPD 레버가 SPEC-032와 충돌.
**오너 결정**: 휴면 복원(1.0) / 고요=등장률 수용 / 기본 스탯서 3그룹 공정성 감사만.

## 3그룹 동일예산 공정성 감사 (hidden=1.0, 실스탯 300/트랙 전트랙)
복원 검증: σ=**3.30** (SPEC-032 1.98~3.81 범위 — 무회귀 확인).
동일 Total 코호트 내 승률 스프레드:

| Total 코호트 | 스프레드 | 평가 |
|------|------|------|
| 90 (최저, 5캐릭) | ~4pp (20.1~24.2%) | 매우 공정 |
| 96 (7캐릭) | ~12pp (17.4~29.5%) | 경미 |
| 100 (5캐릭) | ~12pp (17.1~29.7%) | 경미 |
| 114 (고요 단독, Core38/Sup36/Var40) | 74.9% | 다중-max 아웃라이어 |

**결론**:
1. **90/10식 동일예산 불공정 없음** — 최대 ~12pp, 최저티어 ~4pp. 분배 압승 구조 아님
2. 승률 지배 = **Total 예산 티어(의도)** + **고요 단독 전스탯 몰빵**(스탯가치 아닌
   개별 캐릭 문제) → **등장률(현 3, 레어)** 레버가 정답(오너 결정)
3. β의 Core 과대가치는 실재하나 **로스터 레벨 불공정으로 미발현**(동일 Total서
   고Core가 체계적 압승 안 함: 홍염 Core37→17%, 심볼릭 Core34→30%). Total이 신호 지배
→ **스탯 비중 재보정 불필요**. 숨은레이어 휴면 유지
