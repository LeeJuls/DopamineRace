## 5. 코드 메모

- `DR_STEAM` 미정의 시: `SteamManager`·`SteamAchievements` 전체가 안전한 no-op 스텁 → 빌드 안 깨짐. 현재 main은 이 상태.
- `SteamManager`는 `RuntimeInitializeOnLoadMethod`로 자동 생성 → 씬/프리팹 수동 배치 불필요.
- API Name은 내부 식별자(비노출)라 StringTable 키 발급 대상 아님. 사용자 노출 이름/설명은 파트너 사이트 측 로컬라이즈(§3).
- 누적 조건(예: 50회 승리) 추가 시 Steam **Stat** 정의 후 `SteamUserStats.SetStat`/진행도 연동 필요 — 본 SPEC 범위 밖.
