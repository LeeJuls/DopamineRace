# SPEC-031 — 지능(Intelligence) 부스트 함정 해소 (Phase 1)

**날짜**: 2026-05-16
**상태**: ✅ Phase 1 메커니즘 구현·결정적 검증 완료 (밸런스 스윕·Phase 2 설명은 후속)
**방식**: 밸런스 디자이너 협업 + Harness 백테스트 게이트
**오너 결정**: 밸런스안=A1, 순서=밸런스 먼저 → 설명 다음

## 배경 / 근본 원인

`RacerController_V4.IsInBurstZone` 에서 지능이 부스트 구간 **폭(width)** 을 연장
(`effectiveEnd = start + (end-start)×(1+intMod)`). 폭↑ → `vmax×1.4` 프레임↑ →
부스트 드레인 ×4 누적↑ → 고지능(흔히 저스태미나: 크리스탈 INT20/STA11) 막판
HP 고갈 → `GetHpSpeedMultiplier` 감속 → **지능=순패널티**, 70~100% 역전도 소실.

## 수정 (A1 — 지능 = 늦은 정밀 킥 + 부스트 절약)

| 파일 | 변경 |
|------|------|
| `Assets/Scripts/Data/GameSettingsV4.cs` | 신규 `v4_intBurstShift`(0~1, 기본 0.5)·`v4_intBurstDrainFloor`(0.4~1, 기본 0.55) |
| `Assets/Scripts/Racer/RacerController_V4.cs` | `IsInBurstZone`: 폭 연장 제거 → 구간 통째 **시점 시프트**(`shift=(end-start)×intMod×v4_intBurstShift`). `ConsumeStaminaV4`: 부스트 드레인 지능 비례 할인(`×burstDrainMul×Lerp(1,floor,clamp01((INT-10)/10))`) |
| `Assets/Scripts/Editor/RaceBacktestWindow.cs` | `SimIsInBurstZoneV4` 시프트 미러 + 드레인 할인 미러 (CLAUDE.md 동시수정 규칙 — 속도/HP 시뮬) |

전부 `GameSettingsV4.asset` 필드로 런타임 튜닝. 롤백=`v4_intBurstShift=0`+`v4_intBurstDrainFloor=1.0`(코드 롤백 불필요).

## 결정적 검증 (Harness — 노이즈 승률 무관 수식 게이트)

Runner 부스트 구간 누적 HP 드레인:

| INT | 수정 전 | 수정 후 | 효과 |
|----|--------|--------|------|
| 20 | 30.2 (**최대**) | 13.9 (**최소**) | 패널티 → 이득 |
| 10 | 25.2 | 25.2 | 기준 불변 |
| 0 | 20.2 (최소) | 25.2 (최대) | 이득 → 페이싱 페널티 |

→ 지능↔HP소모 관계 **단조 역전** + 부스트가 결승 쪽으로 시프트. 함정 제거 입증.
컴파일 0 에러(강제 재컴파일 후), 미러 정상.

## 범위 외 발견 (별도 SPEC 권고 — 미해결)

40회/트랙 백테스트(S0·postfix 동일): **임팩트/황금호 57~90% 독식, 평균순위 편차~6,
다수 0%**. 크리스탈/아마존 0%는 지능 함정 외에 **저STA(11) 기반 저HP + 전반적
캐릭터·스탯 불균형**이 주원인 — 지능 수정과 무관하며 S0에도 동일 존재.
→ SPEC-031 범위 밖. **대규모 캐릭터/스탯 밸런스 패스(전용 세션)** 필요.

## S2 경량 종결 (2026-05-17)

원 S2 풀 스윕은 SPEC-032 타입 매크로 전면 재튜닝(burstDrainMul 4→3, 4타입 구간 재배치)으로
전제가 무효화됨: `v4_intBurstShift/DrainFloor`가 asset YAML 미직렬화(코드기본값만),
`equalStats`가 INT를 상수화(지능효과 측정 불가), equalStats OFF는 개별캐릭 불균형이
승률 지배 → 미세 스윕 측정 무의미. **오너 결정: 경량 재검증+동결**(정밀 스윕은
개별 캐릭터 밸런스 SPEC 이후).

- **직렬화**: `v4_intBurstShift: 0.5`·`v4_intBurstDrainFloor: 0.55` GameSettingsV4.asset에
  영속(코드 변경 0, 런타임 동작 불변) → 향후 .asset 편집 스윕 가능
- **결정적 재검증**(SPEC-032 매크로 실값, Runner 0.08–0.36/burstDrainMul3/floor0.55):
  INT20 누적드레인 20.8(×1.65)·shift +0.028(결승쪽) / INT10·INT0 37.8(×3.0) →
  고INT<저INT **단조 유지 = 함정 역전 새 매크로서도 보존** ✅
- **타입 σ 회귀 가드**(실스탯 300/트랙): 도주21.6·선행23.2·선입27.7·추입27.6,
  **σ=2.67** (SPEC-032 S3 1.98~S4 3.81 범위·≤5) → 직렬화가 타입 밸런스 불교란 ✅
