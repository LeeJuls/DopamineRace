# spec036 — HONETi 픽셀 GUI 부분 적용 (리더보드)

**날짜**: 2026-06-03 | **상태**: 구현 완료
**범위**: 레이스 후 화면 중 **리더보드(Top 100)**. 메인 베팅·결과 화면 제외.

## 배경
- HONETi Pixel Art Full GUI Kit 구매·임포트 (`Assets/Honeti/PixelArtGUI/`).
- **부분 적용 판단**: HONETi 프리팹은 TMP + VT323(라틴 전용, 한글 없음) → 전체 교체 시 한글 깨짐 + 작동 로직 폐기. **9-slice 스프라이트만** 활용, 텍스트는 기존 legacy Text + 한글 폰트 유지.
- 적용 대상 선별: **Result는 이미 크림 패널로 스킨돼 효과 미미 → 제외**. 리더보드는 단색 박스 → HONETi `RankingScore` 디자인과 정확히 일치 → 채택.
