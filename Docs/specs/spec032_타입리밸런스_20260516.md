# SPEC-032 — 타입 구조 매크로 리밸런스 (독식/0% 해소)

**날짜**: 2026-05-16
**상태**: ✅ 타입 매크로 동결 (후속 3개 중 #1)
**방식**: 밸런스 디자이너 + Harness 백테스트, GameSettingsV4.asset만 (CSV 불변 — 오너 결정)

## 배경 / 근본 원인

전 트랙에서 도주(Runner) 전 캐릭터 0%, 선행~0%, 선입/추입 독식, 편차~6.
원인 = **타입 부스트 종료 후 결승스퍼트(0.80)까지 "느린 Normal(dead stretch)" 길이 차이**.
도주 0.56(최악) vs 추입 0.00(스퍼트 직결) → 스탯 무관 타입 구조 문제.
equalStats ON S0에서 추입47%/선입36%/도주9%/선행7.5% (추입+선입 83%) → 100% 타입구조 입증.

`RaceBacktestWindow` 시뮬은 asset 값을 직접 읽으므로 미러 코드 변경 불필요(SPEC-031 미러는 지능 한정).
