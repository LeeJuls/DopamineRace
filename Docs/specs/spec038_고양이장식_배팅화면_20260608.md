# SPEC-038 — 배팅 화면 배경 고양이 장식 (Pet Cats)

**날짜**: 2026-06-08
**상태**: 구현 완료
**관련 히스토리**: `Docs/history/20260608_고양이장식_히스토리.md`

## 배경
Pet Cats Pixel Art Pack의 고양이를 배팅 화면 배경(좌하단 모래밭)에 띄워 생동감을 준다. 경기 시작 전에만 등장하고 경기 중에는 사라지는 분위기 연출.

## 요구사항
- **R1** `GameState.Betting`에서만 고양이 N마리가 사용자 지정 폴리곤 영역 안을 배회.
- **R2** 경기 시작(Countdown/Racing) 시 despawn, 배팅 복귀 시 재등장.
- **R3** 고양이 종류는 6종 중 **가중치 랜덤** N종(비복원).
- **R4** 고양이별 **등장확률(setting)** 을 디자이너가 조정 — 별도 ScriptableObject.
- **R5** 배회 영역은 **에디터에서 폴리곤 꼭짓점을 직접 찍어** 지정(WaypointEditor 방식).
- **R6** 동시 등장 마리수(spawnCount)도 setting으로 조정.
- **R7** 기존 racer 배회·레이스·결과에 회귀 영향 0. UI 문자열 없음(StringTable 무관).

## 설계 — 신규 7파일 + 에셋(SO1·프리팹6) + 배선 2줄
| 파일 | 역할 |
|------|------|
| `Data/CatAreaData.cs` | 폴리곤 영역 JSON(`StreamingAssets/cat_area.json`). `Contains`(ray-casting)·`RandomPointInside`(bbox rejection 30회+중심 clamp)·`GetBounds`. TrackPathData 패턴 |
| `Data/CatSpawnConfig.cs` (SO) | 종별 가중치(`cats[]`)·`spawnCount`·`frameInterval`. `PickWeightedDistinct(n)`·`OnValidate` 경고. `Resources.Load`+Instance 싱글턴 |
| `Utility/WeightedRandomHelper.cs` | `PickIndex`(룰렛휠)·`PickDistinct`(비복원, 가중0 제외). CharacterData 비의존(단위테스트용) |
| `Racer/CatAreaEditor.cs` (#if EDITOR) | **C**토글·드래그·**+/−** 정점·**K**저장. 청록 마커 + LineRenderer. 저장키 K(=S 충돌 회피) |
| `Racer/CatController.cs` | 코드 프레임순환(Animator 미사용)·폴리곤 배회(Moving/Pausing)·**flipX**(pivot 점프 회피). 코루틴 OnDisable 정지 |
| `Manager/CatDecorationManager.cs` (싱글턴) | `OnStateChanged` 구독→Betting spawn/그외 despawn. **종별 풀링(≤6)**·초기보정·OnDestroy 구독해제 |
| `Editor/CatPrefabFactory.cs` (#if EDITOR) | `[MenuItem]` 프리팹6+SO 생성. 시트 sub-sprite **`_정수` 숫자정렬**·`Resources/Cats` 생성·SO idempotent |

배선(`SceneBootstrapper.Awake`): Track 블록에 `CatAreaEditor`, 매니저 블록에 `CatDecorationManager`.

## 등장확률 (CatSpawnConfig.asset)
- `cats[6]` 각 `catNumber`(1~6) + `spawnWeight`(0~100, 0이면 미등장).
- `PickWeightedDistinct(spawnCount)` → 가중치 비복원으로 서로 다른 종 선택. 유효종<spawnCount면 가능분만, 합계0이면 0마리.
- `spawnCount`(0~6), `frameInterval`(0.02~1) 디자이너 조정.

## 비고
- pivot(0,0): Contains 기준 = 고양이 `transform.position`(발). 시각 중심은 flipX로 안정.
- sortingOrder −20(배경 −100 < 고양이 < racer ~0 < UI 100). 영역은 사용자가 C키로 재배치 가능.
- Android: `StreamingAssets` `File.ReadAllText` 데스크톱 한정(기존 waypoint와 동일 제약).
