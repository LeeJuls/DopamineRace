## 배경

SampleScene(배팅/레이싱 화면)에서 ESC 키로 열리는 옵션 패널 구현.  
BGM/SFX 음소거 + 재시작/종료(확인 팝업) 기능. 모두 프리팹 기반, 7개 언어 다국어.

---

## 요구사항

### R1. ESC 키 → 옵션 패널 토글
- SampleScene 어느 상태에서든 ESC 키로 OptionPanel 열기/닫기

### R2. BGM/SFX 음소거 토글
- Toggle UI — ON 시 해당 오디오 무음
- 앱 재시작 후에도 상태 유지 (PlayerPrefs)

### R3. 재시작
- 버튼 → 확인 팝업 → 예: `GameManager.StartNewGame()`
- 아니오: 팝업만 닫힘

### R4. 게임 종료
- 버튼 → 확인 팝업 → 예: `Application.Quit()`

### R5. 프리팹 기반
- `OptionPanel.prefab` — BettingPanel과 완전히 독립
- Editor 스크립트로 자동 생성 (`DopamineRace > Create Option Panel Prefab`)
- `GameSettings.optionPanelPrefab` 레퍼런스, 런타임 Instantiate

### R6. 다국어 (7개 언어)
- StringTable `str.ui.option.*` 키 9개 — 하드코딩 금지

## 설계

### 하네스 구조 (스프린트 계약)
Sprint 0 → Sprint 1 → Sprint 2 → Sprint 3  
각 스프린트 완료 후 QA 에이전트 게이트 통과 시 다음 진입.

### 프리팹 계층
```
OptionPanel (OptionPanel.cs)          ← Canvas 자식으로 Instantiate 필수
  ├── DimBg (black 50%)
  ├── PanelBg (510×600)               ← v2: 340×400 → 510×600
  │     ├── TitleLabel (font 34)
  │     ├── BGMToggle (420×60)
  │     ├── SFXToggle (420×60)
  │     ├── Divider
  │     ├── RestartButton (360×66)
  │     ├── QuitButton (360×66)
  │     └── CloseButton (54×54)
  └── ConfirmPopup (ConfirmPopup.cs, 초기 비활성)
        ├── DimBg (black 70%)
        └── PopupBg (450×290)         ← v2: 300×180 → 450×290
              ├── MessageLabel (Wrap) ← v2: HorizontalOverflow=Wrap
              ├── YesButton (165×54)

```
