# SPEC-048 번체(繁體) 중국어 추가 + 중국어 픽셀 폰트

> 2026-06-27 · 8번째 언어 tw 추가 · TitleScene 선택 · Fusion Pixel 폰트(OFL)

## 1. 배경

기존 7개 언어(ko·en·jp·cn·de·es·br), `cn`=간체(简体). 번체 시스템 언어도 `cn`으로 합쳐져 대만·홍콩 유저가 간체를 봄. 번체(`tw`) 추가 + TitleScene 선택 가능하게.
추가로, 픽셀 폰트(PFstardust)에 중국어 글리프가 없어 cn·tw가 시스템 폰트로 폴백 → 중국어 전용 픽셀 폰트 도입.

## 2. 언어 코드

| 코드 | 의미 | CSV 컬럼 |
|------|------|----------|
| cn | 간체(简体) | index 4 (기존) |
| **tw** | **번체(繁體, 대만)** | **index 8 (신규)** |

## 3. 코드 변경

| 파일 | 변경 |
|------|------|
| `Utility/Loc.cs` | `langIndex`에 `tw=8`. `DetectSystemLanguage`: ChineseTraditional→`tw` 분리. |
| `Manager/TitleSceneManager.cs` | `langCodes`/`langDisplayCodes`에 `tw`/`TW` 삽입(cn 뒤). 드롭다운 자동 확장. |
| `Utility/FontHelper.cs` | `GetMainFont()`가 cn→간체폰트·tw→번체폰트 라우팅(Resources.Load 캐시). `GetFont`/`GetFontForCurrentLanguage`도 경유. |
| `Resources/Data/StringTable.csv` | `tw` 컬럼 423행 추가(대만식 번체, cn 기반 변환). |

## 4. 폰트 — Fusion Pixel Font

| 항목 | 내용 |
|------|------|
| 출처 | github.com/TakWolf/fusion-pixel-font (12px proportional ttf) |
| 라이선스 | **SIL OFL 1.1** (상업적 무료) — `Resources/Fonts/OFL.txt` 동봉 |
| 파일 | `Resources/Fonts/fusion-pixel-12px-proportional-zh_hans.ttf`(cn) · `...zh_hant.ttf`(tw) |
| 임포트 | Dynamic + HintedRaster(픽셀 크리스프) |
| 매핑 | cn→zh_hans, tw→zh_hant. ko·en 등 기존 언어는 PFstardust 유지 |
| **설정** | **`GameSettings.chineseSimplifiedFont`(cn)·`chineseTraditionalFont`(tw) — 인스펙터에서 교체 가능. 미지정 시 Resources 기본 폰트 폴백** |

## 5. 검증 (Unity MCP, 기능)

| 항목 | 결과 |
|------|------|
| 컴파일 에러 | 0 |
| CSV 행 무결성 | 422행 전부 9컬럼, 따옴표 행 정상 |
| StringTable 누락 키 | 0 |
| tw 텍스트 로드 | 점击→點擊, 寂静→寂靜 등 번체 정상, cn과 구분 |
| 폰트 라우팅 | tw=zh_hant, cn=zh_hans, ko/en 회귀 정상 |
| 번체 글리프 | 寂·靜·繁 폰트 존재 확인(두부 없음) |

## 6. 리스크/메모
- 번체는 LLM 변환(대만 어휘 반영) — **출시 전 네이티브 검수 권장**.
- 픽셀 폰트 비네이티브 크기(UI 40·48px) 렌더는 약간 소프트할 수 있으나 글리프는 정상.
- Fusion Pixel은 CJK(일·한 포함) 커버 — 향후 jp 폰트 개선에도 재활용 가능.
