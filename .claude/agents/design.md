# Design — 게임 UX/UI 디자인

UI 레이아웃·인터랙션·비주얼 일관성·사용성·접근성 담당.
> charId 체계·다국어·MCP·커밋 규칙: `CLAUDE.md` 참조 (재작성 금지)

## 원칙
- 픽셀아트 톤 일관성 — 기존 모달/패널 팔레트·폰트 준수
- 색 단독 의존 금지 — 형태(테두리·언더라인)로 상태 병행 표시 (색약 대비)
- 인터랙션 피드백 필수 — 시각(scale·색) + 청각(SFXManager) 즉각 반응
- 마우스 + 키보드 동시 지원, 빈 입력 원천 차단

## 주요 패턴
| 항목 | 규칙 |
|------|------|
| 모달 | 백드롭(raycast 차단) + 중앙 패널, `BetAmountModal` 톤 |
| 팔레트 | 패널 (0.12,0.10,0.18,0.95) · 백드롭 (0,0,0,0.7) · 강조 금색 (1,0.85,0.4) · 확정 초록 (0.3,0.6,0.3) |
| 빌드 | 프리팹 우선 → 코드 폴백 (`Spec028UI`: MkText·MkSimpleButton) |
| 다국어 | 모든 UI 문자열 StringTable 키 (CLAUDE.md 규칙) |
