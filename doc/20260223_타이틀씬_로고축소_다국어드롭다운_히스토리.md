# 20260223 타이틀씬 로고 축소 + 다국어 드롭다운 UI 히스토리

## 작업 개요
타이틀 씬에서 로고 크기 축소, 언어 선택 UI를 드롭다운 방식으로 변경, 7개 언어 지원 추가

## 변경 파일
| 파일 | 변경 내용 |
|------|----------|
| `Assets/Scripts/Manager/TitleSceneManager.cs` | LOGO_SCALE 0.75f, 언어 드롭다운 UI 전체 재설계 |
| `Assets/Scripts/Utility/Loc.cs` | 7개 언어 지원 (cn, de, es, br 추가), 시스템 언어 감지 확장 |
| `Assets/Resources/Data/StringTable.csv` | 전체 재작성 — 8컬럼(UID + 7언어), 154개 키 |

## 상세 변경

### 1. 로고 크기 축소
- `LOGO_SCALE`: `1.0f` → `0.75f` (25% 축소)

### 2. 언어 선택 UI 드롭다운 전환
**기존**: 3개 언어 버튼(한국어/EN/日本語)이 우하단에 나란히 표시
**변경**: 현재 언어코드(예: KR)를 보여주는 메인 버튼 1개 → 클릭 시 7개 언어 드롭다운 표시

#### 구현 상세
- **메인 버튼**: 우하단 (-30, 30), 80×40, 현재 언어 display code 표시
- **드롭다운 패널**: VerticalLayoutGroup + ContentSizeFitter, 배경 Color(0.1, 0.1, 0.1, 0.9)
- **드롭다운 버튼**: 7개, 각 68×36, 언어코드 표시 (BR, ES, DE, CN, EN, JP, KR)
- **외부 클릭 닫기**: 전체화면 투명 오버레이(langOverlay)로 구현, Color(0,0,0,0.01f)
- **Canvas**: sortingOrder 200, ScreenSpaceOverlay

#### 언어 코드 순서 (위→아래)
```
langCodes = { "br", "es", "de", "cn", "en", "jp", "ko" }
langDisplayCodes = { "BR", "ES", "DE", "CN", "EN", "JP", "KR" }
```

#### 새로 추가된 메서드
- `CreateDropdownButton()` — 개별 드롭다운 버튼 생성
- `ToggleLangDropdown()` — 메인 버튼 클릭 시 토글
- `OpenLangDropdown()` / `CloseLangDropdown()` — 열기/닫기
- `UpdateLangMainBtnText()` — 메인 버튼 텍스트 갱신

### 3. Loc.cs 7개 언어 지원
#### langIndex 딕셔너리 확장
```csharp
{ "ko", 1 }, { "en", 2 }, { "jp", 3 },
{ "cn", 4 }, { "de", 5 }, { "es", 6 }, { "br", 7 }
```

#### DetectSystemLanguage() 확장
| SystemLanguage | 반환 코드 |
|---|---|
| Chinese / ChineseSimplified / ChineseTraditional | "cn" |
| German | "de" |
| Spanish | "es" |
| Portuguese | "br" |

### 4. StringTable.csv 전체 재작성
- **헤더**: `UID,ko,en,jp,cn,de,es,br`
- **154개 키** 모든 7개 언어 번역 완료
- 추가 언어 번역 대상: 중국어 간체(cn), 독일어(de), 스페인어(es), 브라질 포르투갈어(br)
